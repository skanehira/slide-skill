# SLIDE-PATTERN: center-illustration-spoke

## Overview

スライド中央にイラスト・人物図・サービスロゴを配置し、その周囲（上・左上・左下・右上・右下）に5〜6個のアイコンを配置。各アイコンから中央へ矢印が向かうスポーク型図解。

## 適したシーン

データ収集・統合・集約の仕組み、複数チャネルから中央プラットフォームへの流入、5〜6種類の入力ソースを示すとき。

## Structure

```yaml
structure:
  layout: hub-spoke-illustration
  center:
    element: illustration-placeholder
    label: サービス名/プラットフォーム名
  spokes:
    count: 5
    positions: [top, top-left, bottom-left, top-right, bottom-right]
    elements: [icon-circle, label]
    arrows: inward
```

## Elements

| 要素 | 配置 | 役割 |
|---|---|---|
| 中央イラスト | 中央（半円形背景付き） | プラットフォーム・サービスを体現する人物・図像 |
| スポークアイコン（上） | 上部中央 | 1つ目のデータソース・入力元 |
| スポークアイコン（左×2） | 左側上下 | 2・3番目のデータソース |
| スポークアイコン（右×2） | 右側上下 | 4・5番目のデータソース |
| 矢印 | 各アイコンから中央へ | データ・情報が中央へ集まることを示す |
| ラベル | 各アイコン下 | データソース名・種別名 |

## Usage Guide

- 中央は人物イラストでなくサービスアイコンやロゴでも成立する
- スポーク数は4〜6が最適。7以上は配置が難しくなる
- 矢印の向きを「外向き」に変えると配布型の図解にも使える
