# SLIDE-PATTERN-donut-left-kpi-bar-right
このファイルはスライドのコンテンツエリア（タイトル行より下の領域）のレイアウトパターン定義書です。タイトルエリアを除いたコンテンツ領域のみを対象とし、スライド全体のテンプレートではありません。

## Overview

左側（40%幅）に大きなドーナツグラフと中央数値を配置し、右側（60%幅）を上下に分割して上段に3つのアイコンKPI、下段に横棒グラフを並べるダッシュボード型レイアウト。ひとつのスライドで複数種類の数値情報を同時に表示したい場合に適している。

- **適用場面:** KPIダッシュボード、業績サマリー、定量レポート
- **情報密度:** 高（グラフ×2種＋KPI×3のコンパクト集約）
- **方向性:** 視覚的な達成率と複数指標の同時提示

## Structure

```
┌─────────────────────────────────────────────────┐
│ タイトルエリア（共通）                            │
├──────────────┬──────────────────────────────────┤
│              │ [KPI1]  [KPI2]  [KPI3]            │
│ ドーナツ      ├──────────────────────────────────┤
│ グラフ        │ ラベルA ████████████  84%          │
│  ( 90% )     │ ラベルB ███████████   72%          │
│              │ ラベルC █████████     60%          │
│  凡例        │ ラベルD ████████      55%          │
└──────────────┴──────────────────────────────────┘
  左:40%              右:60%
```

## Elements

### 左エリア（ドーナツグラフゾーン）
| 要素 | 説明 |
|------|------|
| エリアラベル | 「ドーナツグラフ」等の小文字ラベル |
| ドーナツSVG | width:140px / height:140px / 2分割配色 |
| 中央数値 | font-size:24px / bold / 達成率や主要値 |
| 凡例行 | 2項目を横並び（font-size:11px） |

### 右上エリア（KPIバンド）
| 要素 | 説明 |
|------|------|
| アイコン円 | 32×32px / background:#F0F0F0 |
| 数値 | font-size:20px / bold |
| 単位 | font-size:12px / color:#888 |
| ラベル | font-size:11px / color:#888 |

### 右下エリア（横棒グラフ）
| 要素 | 説明 |
|------|------|
| エリアラベル | 棒グラフのカテゴリ名 |
| ラベル列 | 幅80px / font-size:11px |
| バー背景 | flex:1 / height:16px / background:#F0F0F0 |
| バー本体 | background:#CCCCCC / 可変幅% |
| 数値テキスト | font-size:11px / 末尾に数値表示 |

## Usage Guide

- 左のドーナツは「全体達成率」など単一の主要KPIを表示するのに最適
- 右上KPIは同じ軸（件数、金額、人数など）の異なる切り口を並べる
- 右下横棒グラフはカテゴリ別内訳や部門比較に使用する
- 凡例は2項目以内に絞り、簡潔に保つ
- 色はグレースケール2値（#CCCCCC / #E8E8E8）を基本とし、実装時にブランドカラーへ差し替える
