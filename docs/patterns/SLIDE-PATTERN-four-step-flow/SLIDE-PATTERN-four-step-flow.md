# SLIDE-PATTERN-four-step-flow

このファイルはスライドのレイアウトパターン定義書です。SLIDE.mdと組み合わせてAIツールに渡すことで、このパターンのスライドを生成できます。

## Overview

**パターン名：** four-step-flow
**概要：** 上部に中央揃えのタイトルを配置し、下部に4枚の等幅カードを横並びにしたフロー図レイアウト。各カードは番号バッジ・ラベル・イラストで構成され、カード間の矢印でステップの流れを示す。
**適したシーン：** プロセス・手順・ワークフローの説明。「〜の進め方」「〜のステップ」のように順序のある内容を視覚的に整理したいときに最適。

## Structure（構造）

スライド全体は上下2段で構成される。上段にタイトルエリア（中央揃え）、下段に4つの等幅カードを横に並べる。カード間には順序を示す矢印（▶）を挟む。各カードは上部に番号バッジ（丸形・アクセントカラー）、中央上にテキストラベル、中央下にイラストを縦に積み上げる構成。

    structure:
      layout: four-column-flow
      columns: 4
      header:
        position: top-center
        type: text
        role: スライド全体のタイトル
      cards:
        count: 4
        width: 約22%（カード間の矢印スペースを除いた等分）
        separator: arrow（▶、カード間に配置）
        card_elements:
          - badge: 番号（1〜4）、円形、上部中央に重ね表示
          - label: ステップ名テキスト、カード上部
          - illustration: イラスト・アイコン、カード下部

## Elements（各要素の役割）

| 要素 | 配置 | 役割 |
|---|---|---|
| タイトル（H1） | スライド上部・中央揃え | スライド全体のテーマ・タイトル |
| カード（×4） | 横並び・等幅 | 各ステップのコンテナ。角丸・ボーダー付き |
| 番号バッジ | 各カードの上端中央（カードに重ねて表示） | ステップの順序を示す番号（1〜4） |
| ステップラベル | 各カード中央上部 | そのステップで行うことの短い説明（1〜2行） |
| イラスト | 各カード中央下部 | ステップの内容を補強するビジュアル |
| 矢印（▶） | カード間（計3箇所） | ステップの進行方向・順序の明示 |

## Usage Guide（AIへの使い方）

このパターンをAIに指示する際のプロンプト例：

> 「SLIDE-PATTERN-four-step-flowのレイアウトで、[スライドのタイトル]というタイトルのスライドを作成してください。4つのステップは①[ステップ1]、②[ステップ2]、③[ステップ3]、④[ステップ4]です。デザインはSLIDE.mdに従ってください。」

**注意点：**
- ステップ数は4固定のパターン。3ステップや5ステップに変える場合はカード数を調整すること。
- イラストがない場合は、カード下部をアイコン（SVG・絵文字）またはカラーブロックで代替できる。
- 番号バッジの色はデザインシステムのアクセントカラーを使用する。
- ステップラベルは1〜2行に収まる短い文言が望ましい（長文は視認性を損なう）。
