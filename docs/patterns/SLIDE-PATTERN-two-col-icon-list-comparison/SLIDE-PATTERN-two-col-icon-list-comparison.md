# SLIDE-PATTERN-two-col-icon-list-comparison

このファイルはスライドのコンテンツエリア（タイトル行より下の領域）のレイアウトパターン定義書です。SLIDE.mdと組み合わせてAIツールに渡すことで、このパターンのスライドを生成できます。タイトルエリア・ページ番号・装飾はSLIDE.mdの `Slide Frame` セクションで定義されるため、このファイルには含みません。

## Overview

**パターン名：** two-col-icon-list-comparison
**概要：** コンテンツエリアを左右2カラムに分割し、各カラムに「カラーアイコン＋テキスト」のリストを縦に並べた比較レイアウト。コンテンツエリア上部にリード文を1〜2行置き、2カラムのそれぞれに見出しと3〜4項目のアイコン付きリストを配置する。
**適したシーン：** 原因と対策の比較、現状と施策の対応、AとBの観点比較など、2つの観点からそれぞれ複数の要素を並べて示したいスライド。

## Structure（構造）

コンテンツエリアの上部にリード文（中央揃えの1〜2行テキスト）を置き、その下を左右2等分のカラムに分割する。各カラムは薄い背景色の角丸ボックスで囲まれ、上部にカラーの見出しテキスト、下部にアイコン（正方形）＋テキストのリストアイテムを縦積みで並べる。左右カラムの間には矢印などの繋ぎ要素を置いてもよい。

    structure:
      layout: two-column
      top_lead: true
      left_col:
        bg: light-colored-box
        header: accent-text
        items: icon-text-rows
      right_col:
        bg: light-colored-box
        header: accent-text
        items: icon-text-rows
      center_connector: optional

## Elements（各要素の役割）

| 要素 | 配置 | 役割 |
|---|---|---|
| リード文 | コンテンツエリア最上部・中央揃え | スライドのテーマや2カラムの文脈を簡潔に説明する1〜2行の導入テキスト |
| カラムボックス | 左右の各カラム | 各グループをまとめる薄い背景色の角丸コンテナ（左右で異なる色調を使うと区別しやすい） |
| カラム見出し | 各カラムボックスの上部・中央揃え | そのカラムの内容を端的に示すラベル。アクセントカラーのフォントで強調する |
| アイコン | 各リストアイテムの左端 | アイテムの内容を視覚的に補足する正方形のアイコン画像またはプレースホルダー |
| アイテムテキスト | アイコンの右横 | 各要素の説明テキスト。1〜2行で収める |
| リストアイテム区切り線 | アイテム間 | 各リストアイテムを細い横線で区切り可読性を高める |
| 中央コネクター | 左右カラムの中間 | 左から右への因果・流れを示す三角矢印や記号（省略可） |

## Usage Guide（AIへの使い方）

このパターンをAIに指示する際のプロンプト例：

> 「SLIDE-PATTERN-two-col-icon-list-comparisonのレイアウトで、社会のデジタル化の流れ（左カラム）と企業が取り組むべきデジタル施策（右カラム）をアイコン付きリストで比較するスライドを作成してください。デザインはSLIDE.mdに従ってください。」

**注意点：**
- 左右カラムは同じ項目数で揃えると見栄えが良い（3〜4項目が最適）
- 各カラムの背景色は左右で異なる色相を使うと対比が明確になる
- アイコンは意味のある絵柄を選び、左右で色調を変えるとカラムの違いが伝わりやすい
- リード文は2行以内に収め、カラムのスペースを圧迫しないようにする
