# SLIDE-PATTERN-section-divider

このファイルはスライドのコンテンツエリア（タイトル行より下の領域）のレイアウトパターン定義書です。SLIDE.mdと組み合わせてAIツールに渡すことで、このパターンのスライドを生成できます。タイトルエリア・ページ番号・装飾はSLIDE.mdの `Slide Frame` セクションで定義されるため、このファイルには含みません。

## Overview

**パターン名：** section-divider
**概要：** チャプター区切りスライド。左端に太い縦線、その右に大きなセクション番号・セクションタイトル・サブタイトルを縦に並べ、余白を活かしたシンプルな構成。
**適したシーン：** プレゼンの章区切り、セクション導入、アジェンダ後の各パートの冒頭スライド

## Structure（構造）

コンテンツエリア全体を縦方向中央揃えで使用する。左端に太い縦線アクセントを置き、その右にセクション番号・タイトル・サブタイトルを縦に並べる。下部にオプションでページ範囲テキストを配置できる。

    structure:
      layout: content-area-centered-vertical
      left-accent:
        type: vertical-line (2–4px)
      main-content:
        alignment: vertical-center
        elements:
          - section-number (large, muted, #AAAAAA)
          - section-title (large, bold, #333)
          - sub-title (small, #666)
      footer-optional:
        - page-range-text (small, #AAAAAA)

## Elements（各要素の役割）

| 要素 | 配置 | 役割 |
|---|---|---|
| 縦線アクセント | コンテンツエリア左端 | セクションの存在感を演出するビジュアルアクセント |
| セクション番号 | 縦線の右・上部 | 「01」「02」など章番号を大きく表示し、構成を明示 |
| セクションタイトル | 番号の下 | 章の主題タイトルを大きく太く表示 |
| サブタイトル | タイトルの下 | 章の内容を一言で補足する説明文 |
| ページ範囲（任意） | コンテンツエリア下部 | この章で扱うページ範囲（例: P.10–15）を小さく表示 |

## Usage Guide（AIへの使い方）

このパターンをAIに指示する際のプロンプト例：

> 「SLIDE-PATTERN-section-dividerのレイアウトで、セクション番号「02」、セクションタイトル「[タイトル]」、サブタイトル「[サブタイトル]」の区切りスライドを作成してください。デザインはSLIDE.mdに従ってください。」

**注意点：**
- Slide Frameのタイトルエリアは表示してもよいが、コンテンツエリアが主役のためシンプルに保つこと
- セクション番号は「01」「02」のようなゼロパディング2桁が定番
- 余白を多めに取りシンプルに仕上げることで、前後スライドとのコントラストが生まれる
- ページ範囲テキストは省略しても構わない
