# SLIDE-PATTERN-three-stage-circle-flow
このファイルはスライドのコンテンツエリア（タイトル行より下の領域）のレイアウトパターン定義書です。タイトルエリアを除いたコンテンツ領域のみを対象とし、スライド全体のテンプレートではありません。

## Overview

3つの大きな円形アイコン（ステージ名ラベル付き）を矢印で横に連結し、各円の下に見出し・サブタイトル・区切り線・箇条書きを配置するステージ型フローレイアウト。プロセスの3段階・3フェーズ・3ステップを順序立てて説明する際に適している。

- **適用場面:** プロセス解説、導入手順、ロードマップ概要
- **情報密度:** 中（段階ごとに3項目の詳細を内包）
- **方向性:** 左から右への時系列的な流れの可視化

## Elements

### ステージ円形アイコン
| 要素 | 説明 |
|------|------|
| 円形コンテナ | width:110px / height:110px / border-radius:50% |
| ボーダー | 2px solid #CCCCCC |
| 背景 | #F0F0F0 |
| アイコン代替文字 | font-size:32px / color:#AAAAAA |
| ステージ名 | font-size:12px / bold / color:#666 |

### テキストエリア（円の下）
| 要素 | 説明 |
|------|------|
| 見出し | font-size:15px / bold / color:#333 / text-align:center |
| サブタイトル | font-size:12px / bold / color:#555 |
| 区切り線 | width:40px / 1px solid #CCCCCC |
| 箇条書き | font-size:11px / color:#666 / 3項目 |

### 矢印
| 要素 | 説明 |
|------|------|
| 矢印文字 | 「→」/ font-size:24px / color:#CCCCCC |
| 配置 | align-self:flex-start / margin-top:48px |
| flex-shrink | 0（矢印が縮まないように固定） |

## Usage Guide

- 3ステージは「現状→移行→目標」「計画→実行→評価」のように対比構造を持たせると効果的
- 円形内のアイコン代替文字は実装時にアイコン画像やSVGアイコンに差し替える
- 各ステージの箇条書きは3項目で統一し、文字数も揃えるとバランスが良い
- 矢印は `align-self:flex-start; margin-top:48px` で円の中央付近に合わせる
- ステージ名はステージ番号（STEP1など）とラベルを組み合わせると識別しやすい
