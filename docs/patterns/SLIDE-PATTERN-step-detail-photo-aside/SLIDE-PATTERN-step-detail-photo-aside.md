# SLIDE-PATTERN: step-detail-photo-aside

## 概要

左列に縦ステップ円（課題→取り組み→成長実感等）、中列に各ステップの見出し＋箇条書き本文、右列にプロフィール写真・氏名・役職を配置した3列構成のケーススタディレイアウト。

## 適したシーン

社員インタビュー・事例紹介・取り組み紹介で、ステップ（課題・施策・結果）と担当者情報を1枚に収めるとき。

## Structure

```yaml
structure:
  layout: three-column
  left_col:
    width: 12%
    element: vertical-step-circles (with dotted connector)
  middle_col:
    width: 63%
    element: step-blocks (heading + bullet-list per step)
  right_col:
    width: 25%
    element: profile (circle-photo + name + title)
```

## Elements

| 要素 | 配置 | 役割 |
|---|---|---|
| ステップ円 | 左列・縦配置（3個） | 各フェーズ名（課題・取り組み・成長実感等）を円で表示 |
| 点線縦連結 | ステップ円の間 | フェーズの流れを示す縦の点線 |
| ステップ見出し | 中列・各ブロック上部 | 各フェーズのメッセージ・小見出し（太字・色付き） |
| 箇条書き本文 | 中列・見出し下 | 具体的な施策・内容・エピソード（2〜3行） |
| 分割線 | 中列・ブロック間 | 各ステップブロックを区切る水平線 |
| 丸型写真 | 右列上部 | 担当者・インタビュイーの顔写真 |
| 氏名 | 写真直下 | 担当者名（大きめ） |
| 役職・所属 | 氏名下 | 英字役職名・部署名 |

## Usage Guide

- 左のステップ円は3個が標準。2個だと間延びする
- 中列の各ステップブロックは均等な高さにする
- 右の写真エリアは縦中央揃えにするとバランスが取れる
- ステップ円の最後（成果・成長実感）を濃い背景色にすると強調効果がある
- 中列の見出しにブランドカラーを適用すると視認性と統一感が上がる
