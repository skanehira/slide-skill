# SLIDE-PATTERN-six-card-two-column

このファイルはスライドのコンテンツエリア（タイトル行より下の領域）のレイアウトパターン定義書です。SLIDE.mdと組み合わせてAIツールに渡すことで、このパターンのスライドを生成できます。タイトルエリア・ページ番号・装飾はSLIDE.mdの `Slide Frame` セクションで定義されるため、このファイルには含みません。

## Overview

**パターン名：** six-card-two-column
**概要：** 6つの項目を2列×3行のカードグリッドに配置し、各カードに番号・見出し・本文を入れるレイアウト
**適したシーン：** 6つの特徴・メリット・リスク・ポイントの説明、リモートワーク・DXなど複数要素の列挙

## Structure（構造）

コンテンツエリアを2列×3行の6カードに均等分割する。各カードは番号＋見出し（上部）と本文（下部）で構成される。カードはボーダーで区切り、左右のカードが同じ行に並ぶ。

    structure:
      layout: two-column-three-row-grid
      columns: 2
      rows: 3
      card:
        elements:
          - number-badge
          - heading
          - body-text

## Elements（各要素の役割）

| 要素 | 配置 | 役割 |
|---|---|---|
| 番号 | 各カード左上 | 1〜6の番号 |
| 見出し | 番号右（太字） | カードのキーワード |
| 本文 | 見出し下 | 詳細説明（2〜3行） |
| カード枠 | 各カード全体 | 6項目の境界を明確にする |

## Usage Guide（AIへの使い方）

このパターンをAIに指示する際のプロンプト例：

> 「SLIDE-PATTERN-six-card-two-columnのレイアウトで、6つの[テーマ]を2列×3行で説明してください。デザインはSLIDE.mdに従ってください。」

**注意点：**
- 6カード固定のパターン。5項目や7項目の場合は調整が必要
- 1カードの本文は2〜3行を超えると詰まるため簡潔に記述する
- 左右2つのカードを対になるペア（例：メリット vs デメリット）にする使い方も有効
