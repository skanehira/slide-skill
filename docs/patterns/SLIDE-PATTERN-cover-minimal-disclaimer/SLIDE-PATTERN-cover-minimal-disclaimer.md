## 概要

白背景ミニマルデザイン。左上にロゴ、中央にタイトル・発表者情報、最下部に免責事項テキスト行を持つ表紙レイアウト。金融・法務資料に適した。

## Elements

| 要素 | 配置 | 役割 |
|---|---|---|
| ロゴ＋会社名 | 左上 | ブランドアイデンティティ |
| メインタイトル | 中央 | 資料・発表のタイトル |
| 発表者名 | タイトル下 | 著者・登壇者情報 |
| 日付・イベント名 | 発表者下 | 発表コンテキスト |
| 免責事項 | 最下部 | 法的免責・金融商品取引業登録情報等 |

## Usage Guide

- 免責事項テキストは金融・証券会社向け資料に必須。2〜3行まで対応
- ロゴエリアの横にサービス名ラインを追加可能
- タイトル下の余白は発表者情報が多い場合に調整可能
- 免責事項フォントサイズは9〜10pxが適切。それ以下は可読性が著しく低下する
- ヘッダー・ボディ・フッターの3セクション構成を崩さないことで情報の階層が明確になる
- 金融規制上必要な登録番号・協会名は省略不可。文字数が多い場合は2行に分割する
