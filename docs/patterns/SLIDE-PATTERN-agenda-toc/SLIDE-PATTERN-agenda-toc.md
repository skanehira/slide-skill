# SLIDE-PATTERN-agenda-toc

このファイルはスライドのレイアウトパターン定義書です。SLIDE.mdと組み合わせてAIツールに渡すことで、このパターンのスライドを生成できます。

## Overview

**パターン名：** agenda-toc
**概要：** 左上にセクションラベルとタイトルを配置し、その下に番号付き目次リストを縦に並べた1カラムレイアウト。各行は大きな番号・縦線・項目名・点線・ページ番号で構成される。
**適したシーン：** プレゼンテーションの目次・アジェンダスライド。章立てのある資料の冒頭や、セクション区切りの案内に最適。

## Structure（構造）

スライド全体は1カラム構成。左上にセクションラベル（小文字大文字・アクセントカラー）とタイトル（H1）を配置し、その下に番号付きリストを縦に等間隔で並べる。各リスト行は左から順に「大番号・縦区切り線・項目名・点線リーダー・ページ番号」で構成され、左右に余白を持たせた中央エリアに整列する。

    structure:
      layout: single-column
      columns: 1
      header:
        position: top-left
        elements:
          - section-label: 小さいセクション名ラベル（例：AGENDA）
          - title: スライドタイトル（例：目次）
      body:
        position: below-header
        type: numbered-list
        items:
          count: 可変（サンプルは6項目）
          row_elements:
            - number: 大きな2桁番号（01〜）、左端
            - divider: 縦線（|）
            - label: 項目名テキスト
            - leader: 点線（……）、項目名とページ番号の間を埋める
            - page: ページ番号（p.XX）、右端

## Elements（各要素の役割）

| 要素 | 配置 | 役割 |
|---|---|---|
| セクションラベル | 左上・タイトルの上 | このスライドの種別を示す小見出し（例：AGENDA） |
| タイトル（H1） | 左上 | スライドのタイトル（例：目次） |
| 番号（01〜） | 各行の左端 | セクションの順序を示す大きな番号 |
| 縦区切り線（\|） | 番号の右隣 | 番号と項目名の視覚的な区切り |
| 項目名 | 縦区切り線の右 | 各セクションの名称 |
| 点線リーダー（……） | 項目名とページ番号の間 | 視線を右端のページ番号へ誘導する |
| ページ番号（p.XX） | 各行の右端 | 対応するページ番号 |

## Usage Guide（AIへの使い方）

このパターンをAIに指示する際のプロンプト例：

> 「SLIDE-PATTERN-agenda-tocのレイアウトで、目次スライドを作成してください。セクションラベルは「AGENDA」、タイトルは「目次」です。項目は①[セクション名 p.XX]、②[セクション名 p.XX]、…としてください。デザインはSLIDE.mdに従ってください。」

**注意点：**
- 番号・項目名・ページ番号はすべてアクセントカラー（デザインシステムの主要カラー）で統一するとまとまりが出る。
- 項目数は4〜8項目程度が視認性の観点から望ましい。それ以上になる場合は2カラム化を検討する。
- 点線リーダーはCSSの `border-bottom: 1px dotted` または `letter-spacing` で再現できる。
- ページ番号が不要な場合は省略し、点線リーダーも削除してよい。
