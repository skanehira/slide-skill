# SLIDE-PATTERN-icon-left-text-list

このファイルはスライドのコンテンツエリア（タイトル行より下の領域）のレイアウトパターン定義書です。SLIDE.mdと組み合わせてAIツールに渡すことで、このパターンのスライドを生成できます。タイトルエリア・ページ番号・装飾はSLIDE.mdの `Slide Frame` セクションで定義されるため、このファイルには含みません。

## Overview

**パターン名：** icon-left-text-list
**概要：** 3〜4つの項目を縦に並べ、各行の左にアイコン（画像・シンボル）、右に見出しと本文を配置する横展開リストレイアウト
**適したシーン：** サービスの機能説明、手順や前提条件の説明、特徴を視覚的に整理したい場面

## Structure（構造）

コンテンツエリアを3〜4行に分割し、各行は左側のアイコンエリアと右側のテキストエリアで構成される。行間は薄い区切り線で分ける。各行の高さは均等。

    structure:
      layout: icon-left-text-rows
      rows: 3
      row:
        left:
          width: 80-100px
          type: icon (image or symbol)
        right:
          elements: [heading-h2 (colored), body-text]
        divider: bottom-border between rows

## Elements（各要素の役割）

| 要素 | 配置 | 役割 |
|---|---|---|
| アイコン | 各行左側 | 項目をシンボルで視覚化 |
| 見出し（H2） | 右側上部（アクセントカラー） | 項目のキーワード |
| 本文 | 見出し下 | 詳細説明（2〜4行） |
| 行区切り線 | 行間 | 項目の境界を明確にする |

## Usage Guide（AIへの使い方）

このパターンをAIに指示する際のプロンプト例：

> 「SLIDE-PATTERN-icon-left-text-listのレイアウトで、3つの[テーマ]をアイコン付きの縦リストで説明してください。デザインはSLIDE.mdに従ってください。」

**注意点：**
- アイコンは実際の画像・SVG・シンボル文字で表現する
- 見出しはアクセントカラー（ブランドカラー）にすると視認性が上がる
- 各行の本文テキスト量を均等に揃えるとバランスが良い
- このパターンはnumbered-list-with-bodyと似ているが、数字バッジの代わりにアイコン画像を使う点が特徴
