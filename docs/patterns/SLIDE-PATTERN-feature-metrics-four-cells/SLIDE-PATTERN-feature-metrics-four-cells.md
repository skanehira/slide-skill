# SLIDE-PATTERN-feature-metrics-four-cells

このファイルはスライドのコンテンツエリア（タイトル行より下の領域）のレイアウトパターン定義書です。SLIDE.mdと組み合わせてAIツールに渡すことで、このパターンのスライドを生成できます。タイトルエリア・ページ番号・装飾はSLIDE.mdの `Slide Frame` セクションで定義されるため、このファイルには含みません。

## Overview

**パターン名：** feature-metrics-four-cells
**概要：** 4つの機能・特徴をカード形式で横並びに配置し、各カード内に機能説明と複数の定量指標（KPI）をセットで見せるレイアウト。
**適したシーン：** 製品・サービスの機能紹介、強みの訴求、導入効果の数値説明など、複数の特長を並列かつ具体的に伝えたいスライド。

## Structure（構造）

コンテンツエリアの上部にリード文（1〜2行）を置き、その下に4つのカードを等幅で横一列に並べる。各カードは上部に番号・アイコン・見出しを持ち、下部にKPI（指標名・数値・単位）を複数行積み重ねる縦構造を取る。

    structure:
      layout: four-column-cards
      lead_text: true          # カード群の直上にリード文を1〜2行配置
      columns: 4
      card:
        header:
          - number_badge       # 01〜04などの連番バッジ
          - icon               # 機能を表すアイコン（画像またはSVG）
          - heading            # 機能名（短いテキスト）
        body:
          - description        # 機能の説明文（2〜3行）
          - metrics_list       # 指標名と数値を交互に並べたリスト（2〜4項目）

## Elements（各要素の役割）

| 要素 | 配置 | 役割 |
|---|---|---|
| リード文 | カード群の上・全幅 | スライド全体の主張を1〜2文で補足する |
| 番号バッジ | 各カード左上 | 機能の順序を示す連番（01, 02…）|
| アイコン | 各カード上部中央 | 機能を視覚的に象徴するイラスト・アイコン |
| カード見出し | アイコンの下 | 機能名を短く端的に表す |
| 説明文 | 見出しの下 | 機能の内容を2〜3行で説明する |
| 指標リスト | カード下部 | 指標名（ラベル）と数値を対で並べて効果を定量化する |

## Usage Guide（AIへの使い方）

このパターンをAIに指示する際のプロンプト例：

> 「SLIDE-PATTERN-feature-metrics-four-cellsのレイアウトで、製品の4つの主要機能とその導入効果の数値を紹介するスライドを作成してください。デザインはSLIDE.mdに従ってください。」

**注意点：**
- カードは必ず4枚で構成する。3枚や5枚にする場合はレイアウト全体のバランスを再調整すること。
- 指標リストの数値は太字・大きめフォントにして視認性を高める。
- 番号バッジは「強調色（アクセントカラー）の背景＋白テキスト」で目立たせると効果的。
- 各カードの指標項目数を揃えると整然として見える（推奨は2〜4項目）。
