# SLIDE-PATTERN-chart-left-text-right

このファイルはスライドのコンテンツエリア（タイトル行より下の領域）のレイアウトパターン定義書です。SLIDE.mdと組み合わせてAIツールに渡すことで、このパターンのスライドを生成できます。タイトルエリア・ページ番号・装飾はSLIDE.mdの `Slide Frame` セクションで定義されるため、このファイルには含みません。

## Overview

**パターン名：** chart-left-text-right
**概要：** 左側にグラフ（折れ線・棒など）、右側にグラフの各系列やポイントの詳細説明を配置する2カラムレイアウト
**適したシーン：** 複数系列のグラフで各系列の意味を右側で解説する場面、データの読み解きを補足したい場面

## Structure（構造）

コンテンツエリアを左右に分割し、左側（約55%）にグラフ、右側（約45%）に凡例＋各系列の説明テキストを配置する。右側の説明は2〜3項目の縦リスト形式（各項目：色ドット＋系列名＋説明文）で構成する。

    structure:
      layout: two-column
      left:
        width: 55%
        type: chart
        elements: [line-or-bar-chart, x-axis, y-axis, legend]
      right:
        width: 45%
        type: text-list
        items: 2-3
        item:
          elements: [color-indicator, series-name (bold), description-body]

## Elements（各要素の役割）

| 要素 | 配置 | 役割 |
|---|---|---|
| グラフ | 左カラム全体 | データの推移や比較を視覚化 |
| グラフ軸ラベル | グラフのX・Y軸 | 期間・単位・目盛りを示す |
| 色ドット/凡例 | 右カラム各項目左端 | グラフの系列色と対応させる |
| 系列名 | ドット右（太字） | 各系列の名称 |
| 説明文 | 系列名下 | その系列の特徴や傾向の解説（2〜4行） |

## Usage Guide（AIへの使い方）

このパターンをAIに指示する際のプロンプト例：

> 「SLIDE-PATTERN-chart-left-text-rightのレイアウトで、左に[グラフ種類]グラフを配置し、右に[系列1名]・[系列2名]・[系列3名]の説明をそれぞれ記述してください。デザインはSLIDE.mdに従ってください。」

**注意点：**
- 右側の説明テキストはグラフの凡例色と一致させること
- 説明が多い場合は3系列が上限（4系列以上は右側が詰まる）
- グラフはあくまで左側に収め、右側を説明エリアとして必ず確保する
