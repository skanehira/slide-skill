## Overview

**パターン名：** three-col-icon-card-with-cta
**概要：** 3列に円形アイコン・強調見出し・説明テキストを並べるカードレイアウト。カード背景は白で、アイコンは円形の枠で強調する。スライド上部に太字のリード文を持つ。
**適したシーン：** サービスの特徴・メリット・提供価値を3点で訴求するスライド。アイコン+数値や短いキャッチコピーをビジュアルで際立たせたいとき。

## Structure（構造）

コンテンツエリアの上部に2段構成のリード文（通常テキスト＋強調テキスト）を置き、その下に3つのカードを横並びで配置する。各カードは円形アイコン・強調見出し（主テキスト）・補足見出し・説明テキストで縦に積み上げられた構造。カードに枠線はなく、アイコン周囲の円形ボーダーが視覚的アクセントとなる。

    structure:
      layout: three-column-grid
      lead_text:
        type: two-line
        line1: plain
        line2: bold-accent
      columns: 3
      card:
        icon_shape: circle
        heading_primary: true
        heading_secondary: true
        description: center-aligned
        background: none

## Elements（各要素の役割）

| 要素 | 配置 | 役割 |
|---|---|---|
| リード文（通常行） | コンテンツ上部 | 前提・状況を説明するテキスト |
| リード文（強調行） | リード文通常行の直下 | スライドの主張・訴求内容を太字・アクセントカラーで表示 |
| 円形アイコン | カード上部中央 | 各カードのテーマを視覚的に示すアイコン（円形ボーダー付き） |
| 主見出し | アイコン直下、中央揃え | カードのキーワード・数値などを強調表示（アクセントカラー） |
| 補足見出し | 主見出し直下、中央揃え | 主見出しを補足する短いテキスト |
| 説明テキスト | 補足見出し下、中央揃え | カードの内容を詳しく説明する2〜3行のテキスト |

## Usage Guide（AIへの使い方）

このパターンをAIに指示する際のプロンプト例：

> 「SLIDE-PATTERN-three-col-icon-card-with-ctaのレイアウトで、サービスで獲得できるリードの3つの特徴（アイコン・数値・説明付き）を訴求するスライドを作成してください。デザインはSLIDE.mdに従ってください。」

**注意点：**
- リード文は2行構成とし、2行目を太字＋アクセントカラーで強調する
- アイコンは大きめの円形ボーダー（直径80〜100px程度）で囲み中央配置する
- 主見出しはアクセントカラーで表示し、数値や短いキャッチコピーに使う
- 補足見出しは主見出しと同色で小さめのフォントにする
- 説明テキストは中央揃えにし、2〜3行に収める
- カードに枠線や背景色は付けず、アイコンの円形ボーダーだけを視覚的アクセントとする
