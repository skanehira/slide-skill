# SLIDE-PATTERN-three-col-kpi-with-chart

このファイルはスライドのコンテンツエリア（タイトル行より下の領域）のレイアウトパターン定義書です。SLIDE.mdと組み合わせてAIツールに渡すことで、このパターンのスライドを生成できます。タイトルエリア・ページ番号・装飾はSLIDE.mdの `Slide Frame` セクションで定義されるため、このファイルには含みません。

## Overview

**パターン名：** three-col-kpi-with-chart
**概要：** 3つのKPI（重要指標）を縦長カードで横並びに配置し、各カード内に大きな数値・補足説明・グラフ（棒グラフ・折れ線グラフ・ドーナツグラフなど）をセットで見せるレイアウト。
**適したシーン：** 市場規模・シェア・成長率など複数の異なる指標を比較・並列提示したい場面。データの根拠をグラフで補足しながらKPIの説得力を高めたいスライド。

## Structure（構造）

コンテンツエリアの上部にリード文（1〜2行）を置き、その下に3つのKPIカードを等幅で横一列に並べる。各カードは上部に大きな数値（KPI値）、中部に補足説明テキスト、下部にグラフ（種類はカードごとに異なってよい）を縦に積み重ねる構造を取る。

    structure:
      layout: three-column-cards
      lead_text: true          # カード群の直上にリード文を1〜2行配置
      columns: 3
      card:
        top:
          - kpi_label          # KPIの名称（見出し）
          - kpi_value          # メインの数値（特大フォント）
        middle:
          - description        # KPI値の補足説明（2〜3行）
          - chart_label        # グラフの見出し
        bottom:
          - chart              # グラフ本体（棒グラフ／折れ線／ドーナツ等）

## Elements（各要素の役割）

| 要素 | 配置 | 役割 |
|---|---|---|
| リード文 | カード群の上・全幅 | スライドの主張を1〜2文で補足する |
| KPIラベル | 各カード上部 | 指標の名称を示す見出し |
| KPI数値 | KPIラベルの下 | 特大フォントで主要な数値を強調表示する |
| 補足説明 | KPI数値の下 | 数値の背景・文脈を2〜3行で説明する |
| グラフ見出し | 補足説明の下 | グラフの内容を一言で示すラベル |
| グラフ本体 | カード下部 | 数値の推移・内訳を視覚化する（種類は自由） |

## Usage Guide（AIへの使い方）

このパターンをAIに指示する際のプロンプト例：

> 「SLIDE-PATTERN-three-col-kpi-with-chartのレイアウトで、市場規模・当社シェア・成長率の3指標をKPIカードとグラフで紹介するスライドを作成してください。デザインはSLIDE.mdに従ってください。」

**注意点：**
- KPI数値は特大フォント（目安：40〜56px相当）で中央揃えにし、視認性を最優先にする。
- 3枚のカードのグラフ種類は統一しなくてよい（例：左＝ドーナツ、中央＝棒グラフ、右＝折れ線）。
- カード全体の高さを揃えるため、グラフエリアの高さを固定して設計する。
- 強調したいカードは枠線や背景色を変えて視覚的に目立たせる（「おすすめ」バッジ相当）。
