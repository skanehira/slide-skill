# SLIDE-PATTERN-faq-three-column-grid

このファイルはスライドのコンテンツエリア（タイトル行より下の領域）のレイアウトパターン定義書です。SLIDE.mdと組み合わせてAIツールに渡すことで、このパターンのスライドを生成できます。タイトルエリア・ページ番号・装飾はSLIDE.mdの `Slide Frame` セクションで定義されるため、このファイルには含みません。

## Overview

**パターン名：** faq-three-column-grid
**概要：** FAQ（よくある質問と回答）を3列×2行のグリッドに整列して配置するレイアウト。各セルは「Q番号バッジ」「質問テキスト（太字）」「回答テキスト」で構成される。
**適したシーン：** よくある質問と回答・導入事例Q&A・製品FAQ・懸念事項への回答スライドなど、複数の質問と答えをコンパクトにまとめたいシーン

## Structure（構造）

コンテンツエリアは3列×2行の均等グリッドで構成される。各セルはボーダー付きカードで、上部中央に「Q番号バッジ（着色背景の小ボックス）」、その下に質問テキスト（太字・やや大きめ）、さらに下に回答テキスト（通常ウェイト・小さめ）が縦に並ぶ。グリッドは左上から右へ、上段→下段の順に並ぶ。

    structure:
      layout: grid
      columns: 3
      rows: 2
      cell:
        elements: [q-badge, question-text, answer-text]
        border: true
        badge-position: top-center

## Elements（各要素の役割）

| 要素 | 配置 | 役割 |
|---|---|---|
| Qバッジ | 各セル・上部中央 | 質問番号を着色背景の小ボックスで表示（例：Q1、Q2） |
| 質問テキスト | 各セル・バッジの下 | 質問内容を太字・やや大きめのフォントで表示 |
| 回答テキスト | 各セル・質問の下 | 回答内容を通常ウェイト・小さめのフォントで表示 |
| セルボーダー | 各セル外周 | カード状の枠線で各Q&Aを視覚的に区切る |

## Usage Guide（AIへの使い方）

このパターンをAIに指示する際のプロンプト例：

> 「SLIDE-PATTERN-faq-three-column-gridのレイアウトで、サービス導入に関するよくある質問を6件まとめてください。デザインはSLIDE.mdに従ってください。」

**注意点：**
- Q番号バッジはSLIDE.mdのアクセントカラーで着色する
- セル数は6個（3列×2行）が最適。4〜5個の場合は空きセルにスペーサーを使用する
- 質問テキストは2〜3行、回答テキストは3〜5行に収まるよう内容を調整する
- セル同士の余白（gap）は均等に取り、全体が詰まって見えないようにする
- タイトルエリアにはスライドの見出し（例：「よくあるご質問と回答」）を入れる
