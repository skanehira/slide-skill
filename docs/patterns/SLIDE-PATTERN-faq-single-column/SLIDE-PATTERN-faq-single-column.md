# SLIDE-PATTERN-faq-single-column

このファイルはスライドのコンテンツエリア（タイトル行より下の領域）のレイアウトパターン定義書です。SLIDE.mdと組み合わせてAIツールに渡すことで、このパターンのスライドを生成できます。タイトルエリア・ページ番号・装飾はSLIDE.mdの `Slide Frame` セクションで定義されるため、このファイルには含みません。

## Overview

**パターン名：** faq-single-column
**概要：** 3つのQ&Aを縦一列に並べ、各質問と回答をひとつのボックスにまとめた1列FAQレイアウト
**適したシーン：** 重要なよくある質問3点の詳細説明、サービス購入前の懸念事項への回答

## Structure（構造）

コンテンツエリアを3行に分割し、各行を1つのQ&Aボックスとして配置する。各ボックスは上部に「Q.」テキスト＋質問文、下部に回答本文を入れる。ボックス間は余白で区切る。

    structure:
      layout: single-column-list
      items: 3
      item:
        type: bordered-box
        top:
          elements: [q-label (colored), question-text]
        bottom:
          elements: [answer-body]

## Elements（各要素の役割）

| 要素 | 配置 | 役割 |
|---|---|---|
| Q.ラベル | 各ボックス上部左 | 質問であることを示すラベル（アクセントカラー） |
| 質問文 | ラベル右 | 1〜2行の質問内容 |
| 回答文 | 質問文下 | 2〜4行の回答テキスト |
| ボックス枠 | 各Q&A全体 | Q&Aの区切りと読みやすさを確保 |

## Usage Guide（AIへの使い方）

このパターンをAIに指示する際のプロンプト例：

> 「SLIDE-PATTERN-faq-single-columnのレイアウトで、3つのよくある質問と回答を縦に並べてください。デザインはSLIDE.mdに従ってください。」

**注意点：**
- 3項目が最適。2項目の場合はボックス高さを大きくしてゆとりを持たせる
- 回答が長くなる場合は文字サイズを少し下げてボックス内に収める
- Q.ラベルをアクセントカラーにするとテキストが多くても質問と回答の区別がつきやすい
