# SLIDE-PATTERN-pricing-comparison-table

このファイルはスライドのコンテンツエリア（タイトル行より下の領域）のレイアウトパターン定義書です。SLIDE.mdと組み合わせてAIツールに渡すことで、このパターンのスライドを生成できます。タイトルエリア・ページ番号・装飾はSLIDE.mdの `Slide Frame` セクションで定義されるため、このファイルには含みません。

## Overview

**パターン名：** pricing-comparison-table
**概要：** 複数プランの料金・仕様を比較できる表形式レイアウト。左列に比較項目ラベルを縦並びで置き、右側に各プランの列を横並びにした格子状のテーブル構造。推奨プランを視覚的に強調できる。
**適したシーン：** SaaSやサービスの料金プラン紹介、プラン選択肢の比較訴求、機能・条件の横断比較が必要なスライド。

## Structure（構造）

コンテンツエリアの上部にサブ見出しとリード文を置き、その下に比較テーブルを配置する。テーブルは左端に「比較項目ラベル列」、右側に「プラン列（3列）」を横並びにした格子構造。各プランの先頭行にはプラン名（および推奨バッジ）と料金を大きく表示する。

    structure:
      layout: comparison-table
      lead_text: true          # テーブルの上にサブ見出し＋リード文
      table:
        row_header: true       # 左端列を比較項目ラベル列として固定
        columns: 3             # プラン数（左ラベル列を除く）
        header_row:
          - plan_name          # プラン名
          - recommended_badge  # 推奨バッジ（オプション）
          - price_large        # 料金（特大フォント）
        body_rows:             # 比較項目（行）の繰り返し
          - row_label          # 行ヘッダー（項目名）
          - cell_value         # 各プランのセル値

## Elements（各要素の役割）

| 要素 | 配置 | 役割 |
|---|---|---|
| サブ見出し | テーブルの上 | プラン選択の主張を短く示す |
| リード文 | サブ見出しの下 | 複数プランを用意した背景・補足説明 |
| 推奨バッジ | 推奨プラン列の先頭上部 | 最も訴求したいプランを「おすすめ」等のバッジで強調 |
| プラン名行 | テーブル先頭行 | 各プランの名称を大きく表示する列ヘッダー |
| 料金セル | プラン名行の下 | 月額料金を特大フォントで表示し、価格差を一目で把握させる |
| 比較項目ラベル列 | テーブル左端列 | 行ごとの比較内容を示す固定ラベル |
| セル値 | 各行・各プラン列 | プランごとの仕様値・説明を記載する |

## Usage Guide（AIへの使い方）

このパターンをAIに指示する際のプロンプト例：

> 「SLIDE-PATTERN-pricing-comparison-tableのレイアウトで、3つの料金プランを比較するスライドを作成してください。デザインはSLIDE.mdに従ってください。」

**注意点：**
- 推奨プランの列は枠線・背景色・上部バッジで他列と明確に差別化する。
- 料金の数字は特大フォント（目安：32〜48px相当）にして視認性を最優先にする。
- 比較項目ラベル列は背景色を付けて表の行を視覚的に区切りやすくする。
- 行数が多い場合はセルの上下パディングを小さくして収まるよう調整する（推奨：5〜8行程度）。
