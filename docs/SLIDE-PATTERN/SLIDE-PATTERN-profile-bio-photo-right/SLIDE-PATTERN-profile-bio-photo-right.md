# SLIDE-PATTERN-profile-bio-photo-right

このファイルはスライドのコンテンツエリア（タイトル行より下の領域）のレイアウトパターン定義書です。SLIDE.mdと組み合わせてAIツールに渡すことで、このパターンのスライドを生成できます。タイトルエリア・ページ番号・装飾はSLIDE.mdの `Slide Frame` セクションで定義されるため、このファイルには含みません。

## Overview

**パターン名：** profile-bio-photo-right
**概要：** 左側に「ラベル＋値」形式のプロフィール項目リストを縦に並べ、右側に円形切り抜きの顔写真と氏名・役職を配置する自己紹介スライドのレイアウト。
**適したシーン：** 登壇者・代表者・メンバー紹介、採用スライドの自己紹介ページ、会社概要スライドの経営者紹介など

## Structure（構造）

コンテンツエリアは「左プロフィールリスト（幅約60%）」と「右フォトエリア（幅約40%）」の2カラム構成。左エリアには氏名・役職・出身地・学歴・経歴などをラベル＋値の行形式で縦に並べる。右エリアには円形プロフィール写真を中央上寄りに配置し、その下に氏名と役職を記載する。

    structure:
      layout: two-column
      left:
        width: 60%
        content: profile-item-list
        item-format: label-value-row
      right:
        width: 40%
        content: photo-and-name
        photo-shape: circle
        text-align: center

## Elements（各要素の役割）

| 要素 | 配置 | 役割 |
|---|---|---|
| プロフィール項目行 | 左カラム・縦並び | ラベル（氏名・役職・出身地・学歴・経歴など）と値を横並びに表示する行 |
| ラベルテキスト | 各行・左側（固定幅） | 項目名を短く表示（例：「氏名」「役職」） |
| 値テキスト | 各行・ラベルの右 | 実際の内容を表示。経歴など複数行になることもある |
| 円形写真 | 右カラム・上寄り中央 | 人物の顔写真を円形にトリミングして表示 |
| 氏名テキスト | 右カラム・写真の下 | 写真の人物の氏名を大きめに表示 |
| 役職テキスト | 右カラム・氏名の下 | 会社名と役職をやや小さめに表示 |

## Usage Guide（AIへの使い方）

このパターンをAIに指示する際のプロンプト例：

> 「SLIDE-PATTERN-profile-bio-photo-rightのレイアウトで、代表取締役の自己紹介スライドを作成してください。デザインはSLIDE.mdに従ってください。」

**注意点：**
- ラベル列の幅は「氏名」「役職」「出身地」など最長ラベルに合わせて固定する（約4〜5文字分）
- 経歴など複数年度にわたる項目は、年度を左揃えでサブ行として記載する
- 右カラムの写真は必ず円形（border-radius: 50%）とし、写真がない場合はプレースホルダーを使用する
- 左カラムには上部に細い罫線（区切り線）を入れるとすっきり見える
- 全体の背景は白またはSLIDE.mdのメインカラーを薄くしたものを使用する
