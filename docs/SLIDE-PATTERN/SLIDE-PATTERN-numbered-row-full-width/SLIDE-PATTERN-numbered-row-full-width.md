# SLIDE-PATTERN-numbered-row-full-width

このファイルはスライドのコンテンツエリア（タイトル行より下の領域）のレイアウトパターン定義書です。SLIDE.mdと組み合わせてAIツールに渡すことで、このパターンのスライドを生成できます。タイトルエリア・ページ番号・装飾はSLIDE.mdの `Slide Frame` セクションで定義されるため、このファイルには含みません。

## Overview

**パターン名：** numbered-row-full-width
**概要：** 番号付きの横長バナー行を縦に並べるリストレイアウト。各行は「番号バッジ＋アイコン」「見出し」「説明文」の3ゾーンで構成され、幅いっぱいに広がる。
**適したシーン：** 事業領域・サービス一覧・ステップ紹介など、4〜6項目を番号付きで等価に並列表示したいとき。

## Structure（構造）

コンテンツエリア全幅にわたる横長バナー行を縦方向に積み重ねる。各行は左端に番号バッジ（正方形またはダイヤ型）とアイコン、中央に項目見出し、右側に説明テキストを配置した3列構成。行間には均等な余白を設ける。

    structure:
      layout: stacked-rows
      row_count: 4
      row_composition:
        left: number-badge + icon
        center: item-heading
        right: description-text
      row_width: full
      gap: equal

## Elements（各要素の役割）

| 要素 | 配置 | 役割 |
|---|---|---|
| 番号バッジ | 各行の左端 | 項目の順序を示す番号（01〜Nなど）を表示する正方形バッジ |
| シェブロン（矢印）アイコン | バッジの右隣 | 視線を右方向へ誘導する装飾的なアイコン |
| 項目見出し | 各行の中央左寄り | 項目名・サービス名など短いラベル（強調表示） |
| 説明テキスト | 各行の右ゾーン | 項目の詳細や補足説明（2〜3行程度） |
| 行バナー背景 | 各行全体 | ブランドカラーの塗り背景で各行を視覚的に区切る |

## Usage Guide（AIへの使い方）

このパターンをAIに指示する際のプロンプト例：

> 「SLIDE-PATTERN-numbered-row-full-widthのレイアウトで、[サービス/事業/ステップ名]を4項目並べてください。デザインはSLIDE.mdに従ってください。」

**注意点：**
- 行数は3〜6行が適切。それ以上になると行の高さが圧迫されて読みにくくなる
- 各行の説明テキストはできるだけ同程度の文字数にそろえると整然に見える
- 番号バッジの背景色は行バナー全体と同系色で統一する（コントラストで番号を際立たせる）
- シェブロンアイコンは必須ではないが、視線誘導として有効なため推奨
