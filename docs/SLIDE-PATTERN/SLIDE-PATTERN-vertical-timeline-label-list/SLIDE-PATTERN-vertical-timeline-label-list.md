# SLIDE-PATTERN-vertical-timeline-label-list

このファイルはスライドのコンテンツエリア（タイトル行より下の領域）のレイアウトパターン定義書です。SLIDE.mdと組み合わせてAIツールに渡すことで、このパターンのスライドを生成できます。タイトルエリア・ページ番号・装飾はSLIDE.mdの `Slide Frame` セクションで定義されるため、このファイルには含みません。

## Overview

**パターン名：** vertical-timeline-label-list
**概要：** 縦方向に並んだステップ・フロー項目を、左にステップラベル（見出し）、右に説明テキストボックスとして配置するタイムライン形式のリストレイアウト。左列のステップ間はラインで繋がれ、各ステップのドットがアクセントカラーで強調される。
**適したシーン：** 手順・フロー説明、プロセス説明、段階的な流れの紹介、参画フローや申込手順など順序のある工程を説明するスライド。

## Structure（構造）

コンテンツエリアを左右2カラムに分割する。左カラムには垂直ラインと丸ドットのタイムラインを配置し、各ドットの右横にステップラベル（太字の見出し）を並べる。右カラムには各ステップに対応する説明テキストを角丸ボックスに入れて縦に並べる。左右の対応する行が同じ高さになるよう揃える。

    structure:
      layout: two-column
      left_col_width: 35%
      right_col_width: 65%
      left:
        type: timeline-dots
        connector: vertical-line
        item: label-text
      right:
        type: stacked-boxes
        item: description-text

## Elements（各要素の役割）

| 要素 | 配置 | 役割 |
|---|---|---|
| 垂直ライン | 左カラム中央に縦貫通 | ステップ間を繋ぎ、時系列・順序の流れを視覚的に示す |
| ステップドット | 垂直ライン上の各行位置 | 各ステップの開始点をアクセントカラーの丸で示す |
| ステップラベル | ドットの右横 | ステップ名・工程名を太字で表示する短い見出し |
| 説明ボックス | 右カラム各行 | 各ステップの詳細説明をやや薄い背景の角丸ボックスに入れて表示する |
| 矢印 | 説明ボックスの左端 | 左カラムのラベルと右カラムの説明ボックスを視覚的に繋げる小さな矢印 |

## Usage Guide（AIへの使い方）

このパターンをAIに指示する際のプロンプト例：

> 「SLIDE-PATTERN-vertical-timeline-label-listのレイアウトで、サービス申込から利用開始までの6ステップのフローをタイムライン形式で説明するスライドを作成してください。デザインはSLIDE.mdに従ってください。」

**注意点：**
- ステップ数は4〜7個が最も読みやすい。それ以上になる場合はスライドを分割する
- 左カラムのラベルは短く（10文字以内が目安）、詳細は右の説明ボックスに書く
- 説明ボックスの高さは内容量に応じて可変にし、左カラムのラベルと中央揃えで対応させる
- 最後のステップのドットの下には垂直ラインが続かないようにする
