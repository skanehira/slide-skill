# SLIDE-PATTERN-sidebar-toc-panel

このファイルはスライドのコンテンツエリア（タイトル行より下の領域）のレイアウトパターン定義書です。SLIDE.mdと組み合わせてAIツールに渡すことで、このパターンのスライドを生成できます。タイトルエリア・ページ番号・装飾はSLIDE.mdの `Slide Frame` セクションで定義されるため、このファイルには含みません。

## Overview

**パターン名：** sidebar-toc-panel
**概要：** スライド全体を左右に二分割し、左側に「INDEX」などのセクション見出しを縦書きまたは横書きで大きく置き、右側に番号付き目次項目を縦に並べる目次専用レイアウト。
**適したシーン：** プレゼンテーションの目次（INDEX）スライド、セクション切り替えのアジェンダ表示、資料の章立て紹介。

## Structure（構造）

コンテンツエリアを左右に分割する。左パネルは背景色を付けてセクション見出し（例：「INDEX」）を中央揃えで大きく表示し、右パネルには番号バッジ＋項目テキストの組み合わせを縦方向に等間隔で並べる。

    structure:
      layout: two-column
      left_panel:
        width: 28%            # 左パネルの幅（目安）
        background: accent    # ブランドカラー等の強調背景
        content:
          - section_label     # 「INDEX」「目次」などのラベルテキスト（縦中央揃え）
      right_panel:
        width: 72%
        content:
          - toc_items         # 番号バッジ＋項目テキストの繰り返し（4〜7項目推奨）

## Elements（各要素の役割）

| 要素 | 配置 | 役割 |
|---|---|---|
| 左パネル背景 | 左側全域 | 強調色の背景でセクション感を視覚的に示す |
| セクションラベル | 左パネル中央 | 「INDEX」「目次」などのラベルを縦中央に大きく表示する |
| 右パネル | 右側全域 | 白またはニュートラル背景で目次項目を並べる |
| 番号バッジ | 各目次項目の左 | 章番号を正方形バッジで表示し、視覚的アクセントとなる |
| 目次テキスト | 番号バッジの右 | 章・セクションの題名を表示する（フォントは大きめ） |

## Usage Guide（AIへの使い方）

このパターンをAIに指示する際のプロンプト例：

> 「SLIDE-PATTERN-sidebar-toc-panelのレイアウトで、プレゼンテーションの目次スライドを作成してください。左パネルに「INDEX」を大きく表示し、右パネルに5つの章タイトルを番号付きで並べてください。デザインはSLIDE.mdに従ってください。」

**注意点：**
- 目次項目数は4〜7項目が視認性のうえで最適。それ以上になる場合はフォントサイズを小さくするか、2列化を検討する。
- 番号バッジは強調色の正方形または角丸正方形にし、テキストを白抜きにする。
- 左パネルのセクションラベルは縦中央揃え・大きめフォント（目安：20〜28px相当）で存在感を持たせる。
- 右パネルの目次項目は等間隔に配置し、上下に余白を設けてゆったりと見せる。
