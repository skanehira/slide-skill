# SLIDE-PATTERN-staircase-roadmap-layered

このファイルはスライドのコンテンツエリア（タイトル行より下の領域）のレイアウトパターン定義書です。SLIDE.mdと組み合わせてAIツールに渡すことで、このパターンのスライドを生成できます。タイトルエリア・ページ番号・装飾はSLIDE.mdの `Slide Frame` セクションで定義されるため、このファイルには含みません。

## Overview

**パターン名：** staircase-roadmap-layered
**概要：** 左下から右上に向かって階段状に積み上がる矢印ブロックで時系列ロードマップを表現し、各ステージの下に複数の累積レイヤー帯（横長テキスト行）を重ねるレイアウト
**適したシーン：** 中長期の成長戦略・ロードマップ・フェーズ別計画を時系列で示したいスライド。各フェーズに紐づく施策や達成目標を階層的に表現したい場合に最適

## Structure（構造）

コンテンツエリアは上部の箇条書き（サマリーポイント）エリアと、下部のビジュアルロードマップエリアで縦に分かれる。ロードマップエリアは、左下から右上に向かって高さが増す4つの台形矢印ブロックを横並びに配置し、各ブロックには年度ラベル・ステップ名・主要テキストを入れる。ブロック群の下には、右方向に伸びる複数の横長帯（累積レイヤー）を積み重ねて全フェーズを貫く施策・目標を示す。

    structure:
      layout: staircase-flow-with-layers
      summary_bullets: top
      staircase_area:
        direction: left-bottom-to-right-top
        step_count: 4
        block_shape: trapezoid-arrow
        year_label: above_each_block
      layer_bands:
        position: below_staircase
        count: 3
        direction: horizontal-right

## Elements（各要素の役割）

| 要素 | 配置 | 役割 |
|---|---|---|
| サマリー箇条書き | コンテンツエリア上部 | スライドの要点を2〜3行の箇条書きで簡潔に提示する |
| 年度ラベル | 各ステアケースブロックの上 | フェーズの開始年度・期間を示すテキストラベル |
| ステアケース矢印ブロック（×4） | 中央ロードマップエリア | ステップ名と主要テキストを持つ台形矢印。左から右に高さが増し進捗・発展を視覚化する |
| ステップ名 | 各ブロック内上段 | フェーズの通称（例：ファーストステップ）を小見出しで表示 |
| フェーズ主要テキスト | 各ブロック内中段 | そのフェーズの主要施策・目標を太字で表示 |
| 累積レイヤー帯（×複数） | ロードマップエリア下部 | 複数フェーズにまたがる横断施策・目標を横長の帯として積み重ねて表示 |

## Usage Guide（AIへの使い方）

このパターンをAIに指示する際のプロンプト例：

> 「SLIDE-PATTERN-staircase-roadmap-layeredのレイアウトで、〔会社名・事業名〕の中長期ロードマップを4フェーズで説明してください。各フェーズに年度と主要施策を入れ、下部に累積目標帯を添えてください。デザインはSLIDE.mdに従ってください。」

**注意点：**
- ステアケースブロックは4つが基本。3つにする場合はブロックの幅比率を調整すること
- 左側ブロックほど高さが低く（過去・現在）、右側ほど高く（未来）なるよう配置することで時間的進行を表現する
- 累積レイヤー帯は下から上へ積み重ねる（最下段が最も広い基盤となる施策を示す）
- 現在地のフェーズを強調色で塗り分けることで「現在位置」を明示できる
