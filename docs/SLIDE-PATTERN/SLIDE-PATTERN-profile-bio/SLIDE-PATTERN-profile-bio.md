# SLIDE-PATTERN-profile-bio

このファイルはスライドのコンテンツエリア（タイトル行より下の領域）のレイアウトパターン定義書です。SLIDE.mdと組み合わせてAIツールに渡すことで、このパターンのスライドを生成できます。タイトルエリア・ページ番号・装飾はSLIDE.mdの `Slide Frame` セクションで定義されるため、このファイルには含みません。

## Overview

**パターン名：** profile-bio
**概要：** 左側に顔写真と氏名キャプション、右側に名前・役職・経歴情報テーブルを配置した自己紹介・プロフィールレイアウト
**適したシーン：** 登壇者紹介、会社代表者紹介、担当者プロフィール

## Structure（構造）

コンテンツエリアを左右2カラムに分割する。左カラム（約35%）に顔写真（正方形または縦長）と名前キャプションを配置し、右カラム（約65%）に名前見出し・役職・各種プロフィール情報をテーブル形式で並べる。

    structure:
      layout: two-column
      left:
        width: 35%
        elements:
          - photo (portrait image)
          - name-caption (small text below photo)
      right:
        width: 65%
        elements:
          - full-name (large heading)
          - title-position (text)
          - profile-table:
              rows:
                - label: 出身地
                - label: 学歴
                - label: 経歴 (multiple rows)

## Elements（各要素の役割）

| 要素 | 配置 | 役割 |
|---|---|---|
| 顔写真 | 左カラム上部 | 人物のビジュアル（信頼感・親近感を高める） |
| 名前キャプション | 写真下 | フリガナ付きの氏名 |
| フルネーム（大） | 右カラム上部 | 大きなフォントで名前を強調 |
| 役職・所属 | 名前下 | 会社名・役職 |
| プロフィールテーブル | 右カラム中〜下 | 出身地・学歴・経歴を構造化して表示 |

## Usage Guide（AIへの使い方）

このパターンをAIに指示する際のプロンプト例：

> 「SLIDE-PATTERN-profile-bioのレイアウトで、[氏名]のプロフィールを表示してください。写真エリアは左に、右側に氏名・役職・経歴テーブルを配置してください。デザインはSLIDE.mdに従ってください。」

**注意点：**
- 顔写真は正方形または縦長の比率を推奨（横長だと顔が歪む）
- 円形クロップ（丸い写真）のバリエーションも一般的（右カラムに写真を置くレイアウトもある）
- 経歴テーブルは「年：内容」の2カラム形式が読みやすい
