# SLIDE-PATTERN-vertical-step-flow

このファイルはスライドのコンテンツエリア（タイトル行より下の領域）のレイアウトパターン定義書です。SLIDE.mdと組み合わせてAIツールに渡すことで、このパターンのスライドを生成できます。タイトルエリア・ページ番号・装飾はSLIDE.mdの `Slide Frame` セクションで定義されるため、このファイルには含みません。

## Overview

**パターン名：** vertical-step-flow
**概要：** 4〜6つのステップを縦方向（上から下）に流れ矢印でつなぎ、各ステップに番号・見出し・説明を配置するフローチャートレイアウト
**適したシーン：** 選考フロー・申込フロー・業務プロセス・手順書など、順番通りに進む流れを示す場面

## Structure（構造）

コンテンツエリア中央（または右寄り）に縦フローを配置する。各ステップは左カラム（番号バッジ＋ラベル）と右カラム（内容テキスト）の2列で構成され、ステップ間に下向き矢印を置く。最終ステップは異なる色（強調）にすることが多い。

    structure:
      layout: vertical-flow
      steps: 4-6
      step:
        left:
          elements: [step-label (STEP 1等)]
          background: filled-color
        right:
          elements: [content-text]
          background: bordered-box
      connector:
        type: down-arrow between steps

## Elements（各要素の役割）

| 要素 | 配置 | 役割 |
|---|---|---|
| ステップラベル | 各ステップ左側 | STEP 1/2/3等の番号とラベル |
| 内容テキスト | 各ステップ右側 | そのステップで行うこと・内容 |
| 下向き矢印 | ステップ間 | 順序と流れを示す |
| 最終ステップ強調 | 最下部 | 最終結果・到達点を強調色で示す |

## Usage Guide（AIへの使い方）

このパターンをAIに指示する際のプロンプト例：

> 「SLIDE-PATTERN-vertical-step-flowのレイアウトで、[フロー名]の[N]ステップを縦に並べて説明してください。最後のステップは強調してください。デザインはSLIDE.mdに従ってください。」

**注意点：**
- 5〜6ステップになると縦方向に詰まるため、各ステップの説明は短くする（10〜20字程度）
- ページ全体の縦をコンパクトに使うため、左右のパディングを広めにとると整って見える
- 矢印を省略してシンプルな縦リスト形式にすることもできる
