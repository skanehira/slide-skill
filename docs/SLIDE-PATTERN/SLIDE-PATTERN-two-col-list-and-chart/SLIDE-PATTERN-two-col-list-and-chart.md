## Overview

**パターン名：** two-col-list-and-chart
**概要：** 上部に概要テキスト（リード文）を置き、その下を左右2カラムに分割する。左カラムにはアイコン付きのリスト項目を縦に並べ、右カラムにはドーナツチャートと凡例・注釈を組み合わせたデータビジュアライゼーションを配置するレイアウト。
**適したシーン：** 市場背景と市場シェアの同時提示・定性リストと定量データの対比・競合状況の説明・ポジショニングスライドなど

## Structure（構造）

コンテンツエリアは「リード文エリア（上部全幅）」と「左リスト（幅約45%）」「右チャートエリア（幅約55%）」の上下＋左右構成。リード文エリアには1〜2文の概要テキストを配置する。左カラムにはカラムタイトルと、アイコン＋見出し＋補足テキストの行を3〜5件縦に並べる。右カラムにはカラムタイトルと、主要指標テキスト（大きな数字）、ドーナツチャート図、凡例リスト、注釈テキストを配置する。

    structure:
      layout: two-column
      top:
        content: lead-text
        width: 100%
      left:
        width: 45%
        content: icon-list
        elements: [column-title, icon-item-rows]
        icon-item:
          - icon
          - item-title
          - item-subtitle
      right:
        width: 55%
        content: chart-area
        elements: [column-title, key-metric, donut-chart, legend, footnote]

## Elements（各要素の役割）

| 要素 | 配置 | 役割 |
|---|---|---|
| リード文 | 上部・全幅 | スライド全体の文脈や主張を1〜2文で説明するテキスト |
| 左カラムタイトル | 左カラム上部 | リストのセクション見出し（例：「市場背景」） |
| アイコン | 左カラム各行・左端 | 各リスト項目を視覚的に区別するアイコンまたはシンボル |
| 項目タイトル | 左カラム各行・アイコン右 | リスト項目の見出し（太字） |
| 項目補足テキスト | 左カラム各行・タイトル下 | 項目の補足説明（小さめフォント） |
| 右カラムタイトル | 右カラム上部 | チャートエリアのセクション見出し（例：「市場シェア」） |
| 主要指標テキスト | 右カラム・チャート左 | 大きなフォントで強調する数値や指標（例：「約42万社」） |
| ドーナツチャート | 右カラム・中央右 | 割合やシェアを視覚化する円形のチャート |
| 凡例リスト | 右カラム・チャート下または左 | チャートの各区分の名称とパーセンテージを示すリスト |
| 注釈テキスト | 右カラム最下部 | データの出所・条件・時点などを小さく記載 |

## Usage Guide（AIへの使い方）

このパターンをAIに指示する際のプロンプト例：

> 「SLIDE-PATTERN-two-col-list-and-chartのレイアウトで、市場成長の背景4点と自社の市場シェアデータを1枚にまとめてください。デザインはSLIDE.mdに従ってください。」

**注意点：**
- 左カラムのアイコンは絵文字・SVGアイコン・色付き丸など簡易なものでよい（デザインはSLIDE.mdに準拠）
- ドーナツチャートはSVGで実装し、自社シェアの区分のみ濃い色で強調する
- 主要指標（大きな数字）は右カラム内でドーナツチャートの左か上に配置する
- 凡例はチャートの下または左に横並びで配置し、カラーボックス＋テキストの形式にする
- 注釈テキストは最下部に極小フォント（10px）で配置する
- 左右カラムはボーダーまたは背景色の違いで視覚的に区切ることが望ましい
