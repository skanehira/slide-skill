# SLIDE-PATTERN-category-toc-simple

このファイルはスライドのコンテンツエリア（タイトル行より下の領域）のレイアウトパターン定義書です。SLIDE.mdと組み合わせてAIツールに渡すことで、このパターンのスライドを生成できます。タイトルエリア・ページ番号・装飾はSLIDE.mdの `Slide Frame` セクションで定義されるため、このファイルには含みません。

## Overview

**パターン名：** category-toc-simple
**概要：** カテゴリ見出し（英語太字大文字＋日本語サブタイトル）と縦線付き箇条書き項目を縦積みしたシンプルな全幅目次レイアウト
**適したシーン：** 項目数が多いシンプルな目次スライド、テキスト情報量を優先したいプレゼン、装飾を排したクリーンな構成

## Structure（構造）

コンテンツエリア全体（全幅）を使って、カテゴリ別の目次リストを縦積みで配置する。各カテゴリは「英語太字大見出し＋日本語サブタイトル」の行と、縦線付きの箇条書き項目群で構成される。装飾画像は持たない。

    structure:
      layout: single
      width: 100%
      sections:
        type: category-section
        repeat: 3〜5カテゴリ
        elements:
          - category-heading (英語・太字・大サイズ)
          - category-subtitle (日本語・小サイズ・右横配置)
          - bullet-list (縦線付き箇条書き)

## Elements（各要素の役割）

※ タイトル（H1）はSLIDE.mdの Slide Frame で定義されるため、この表には含めない。コンテンツエリアの要素のみを記述する。

| 要素 | 配置 | 役割 |
|---|---|---|
| カテゴリ見出し（英語・太字） | 各カテゴリの先頭行・左寄せ | セクション名を英語の大きな太字フォントで表示（例：Company, Business, Work Environment） |
| カテゴリサブタイトル（日本語） | カテゴリ見出しの右横 | カテゴリの内容を日本語で補足（細めフォント・灰色系） |
| 縦線 | 箇条書き項目の左端 | 各項目を縦に繋ぐ細い縦線。複数項目のグループ感を表現 |
| 箇条書き項目 | 縦線の右・インデント | 各カテゴリに属するトピック名（丸ポイント付き） |

## Usage Guide（AIへの使い方）

このパターンをAIに指示する際のプロンプト例：

> 「SLIDE-PATTERN-category-toc-simpleのレイアウトで、[カテゴリ名（英語）・日本語サブタイトル・各項目リスト]を使って目次スライドを作成してください。デザインはSLIDE.mdに従ってください。」

**注意点：**
- カテゴリ見出しの英語フォントは太め（font-weight: 700〜900）にすることで視認性が上がる
- カテゴリ数が増える場合（5個以上）は、各カテゴリの項目数を減らして全体の高さに収める
- [[category-toc-with-image]]との違いは装飾画像がない点のみ。右スペースが空く場合はカラーブロックで代替できる
- カテゴリ間の余白を確保することで階層構造が明確になる
