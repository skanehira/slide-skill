# SLIDE-PATTERN-icon-left-text-list-detailed

このファイルはスライドのコンテンツエリア（タイトル行より下の領域）のレイアウトパターン定義書です。SLIDE.mdと組み合わせてAIツールに渡すことで、このパターンのスライドを生成できます。タイトルエリア・ページ番号・装飾はSLIDE.mdの `Slide Frame` セクションで定義されるため、このファイルには含みません。

## Overview

**パターン名：** icon-left-text-list-detailed
**概要：** 縦に3行並ぶリスト形式のレイアウト。各行は左側に大きめのアイコン、右側にアクセントカラーの見出し＋複数行の説明テキストを配置する。行全体に薄いグレーの背景を適用する。
**適したシーン：** サービスの特長・機能・提供内容を詳しく説明するスライド。各項目に説明テキストが長い場合や、アイコンと詳細テキストをセットで見せたいとき。

## Structure（構造）

コンテンツエリアに3行のリストを縦に並べる。各行はカード形式（薄いグレー背景）で構成され、左側に大きめのアイコンエリア、右側に見出しと説明テキストの2段構成を持つ。行間にスペースを設けて視認性を確保する。

    structure:
      layout: vertical-list
      rows: 3
      row:
        icon: left-large
        heading: accent-color
        description: multi-line
        background: tinted-card

## Elements（各要素の役割）

| 要素 | 配置 | 役割 |
|---|---|---|
| アイコン | 各行の左側、中央揃え | 各項目のテーマを視覚的に示す大型アイコン（色付き） |
| 見出し | アイコン右、上部 | 各項目の主題を示す短いテキスト（アクセントカラー・太字） |
| 説明テキスト | 見出しの直下 | 各項目の詳細説明を2〜3行で記述（通常テキスト） |
| 行背景カード | 行全体 | 薄いグレー背景で各行を視覚的に区別する |

## Usage Guide（AIへの使い方）

このパターンをAIに指示する際のプロンプト例：

> 「SLIDE-PATTERN-icon-left-text-list-detailedのレイアウトで、サービスの3つの特長（各特長のアイコン・見出し・詳細説明付き）を紹介するスライドを作成してください。デザインはSLIDE.mdに従ってください。」

**注意点：**
- アイコンは左側に60〜80px程度の大きさで配置し、縦方向は中央揃えにする
- 見出しはアクセントカラー（ブランドカラー）の太字で表示する
- 説明テキストは2〜4行程度で、各項目の内容を具体的に説明する
- 各行に薄いグレー背景（カード形式）を適用し、行間にマージンを設ける
- 全3行の高さを均等にする
- アイコンと右側テキストエリアは縦方向に中央揃えにする
