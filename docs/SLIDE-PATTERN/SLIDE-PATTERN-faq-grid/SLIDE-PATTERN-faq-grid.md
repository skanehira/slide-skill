# SLIDE-PATTERN-faq-grid

このファイルはスライドのコンテンツエリア（タイトル行より下の領域）のレイアウトパターン定義書です。SLIDE.mdと組み合わせてAIツールに渡すことで、このパターンのスライドを生成できます。タイトルエリア・ページ番号・装飾はSLIDE.mdの `Slide Frame` セクションで定義されるため、このファイルには含みません。

## Overview

**パターン名：** faq-grid
**概要：** 6つのよくある質問（Q&A）を3列×2行のグリッドに配置するレイアウト。各セルに番号・質問・回答を収める
**適したシーン：** よくある質問ページ、サービス説明の補足Q&A、懸念事項への回答

## Structure（構造）

コンテンツエリアを3列×2行のグリッドに分割する。各セルは上部に番号バッジ＋質問見出し、下部に回答本文を配置する。セルは境界線またはボックスで区切る。

    structure:
      layout: three-column-two-row-grid
      columns: 3
      rows: 2
      cell:
        elements:
          - number-badge (Q1〜Q6)
          - question-heading
          - answer-body

## Elements（各要素の役割）

| 要素 | 配置 | 役割 |
|---|---|---|
| 番号バッジ | 各セル上部 | Q1〜Q6の番号を示す |
| 質問見出し | バッジ右または下 | 質問文（1〜2行） |
| 回答本文 | 見出し下 | 回答テキスト（2〜4行） |
| セル区切り | セル間 | 各Q&Aの範囲を明確にする |

## Usage Guide（AIへの使い方）

このパターンをAIに指示する際のプロンプト例：

> 「SLIDE-PATTERN-faq-gridのレイアウトで、6つのよくある質問と回答を3列×2行のグリッドで配置してください。デザインはSLIDE.mdに従ってください。」

**注意点：**
- 各セルのテキスト量が均等になるよう質問・回答の長さを揃える
- 6項目が適切。4項目の場合はtwo-column-split-boxesを2行、8項目以上は別パターンを検討
- 質問文は疑問形で短く（20字程度）、回答は3〜4行に収める
