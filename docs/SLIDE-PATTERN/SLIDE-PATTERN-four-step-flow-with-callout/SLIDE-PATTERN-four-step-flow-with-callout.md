# SLIDE-PATTERN-four-step-flow-with-callout

このファイルはスライドのコンテンツエリア（タイトル行より下の領域）のレイアウトパターン定義書です。SLIDE.mdと組み合わせてAIツールに渡すことで、このパターンのスライドを生成できます。タイトルエリア・ページ番号・装飾はSLIDE.mdの `Slide Frame` セクションで定義されるため、このファイルには含みません。

## Overview

**パターン名：** four-step-flow-with-callout
**概要：** 4つのステップカードを横並びに配置し、右端に補足情報・注意事項のコールアウトボックスを2段で積み重ねたフロー図レイアウト
**適したシーン：** 手順・操作ステップの説明スライドで、プロセスの流れに加えて注意事項や補足情報を同時に伝えたい場合

## Structure（構造）

コンテンツエリアは、メインリードコピー（大見出し）エリアを上部に置き、その下に「4ステップカード＋コールアウト列」の横並びレイアウトで構成される。4つのステップカードはそれぞれ番号バッジ・ステップ名・アイコン・説明文を持ち、矢印でつながれる。右端のコールアウト列には2つのボックス（補足・注意）を縦に並べる。最下部には連携要素を示すタイムライン（横線＋アイコン）を置く。

    structure:
      layout: flow-with-sidebar
      main_lead: top
      flow_area:
        direction: horizontal
        step_count: 4
        connector: arrow
      callout_column:
        position: right
        box_count: 2
        types: [supplement, caution]
      timeline_bar: bottom

## Elements（各要素の役割）

| 要素 | 配置 | 役割 |
|---|---|---|
| メインリードコピー | コンテンツエリア最上部 | スライドの主張・結論を大きな見出しで提示する |
| ステップカード（×4） | 中央エリア横並び | 番号バッジ・ステップ名・アイコン・説明文を持つ手順カード |
| ステップ番号バッジ | 各カード上部 | 円形バッジに番号を表示してステップ順序を示す |
| ステップ接続矢印 | カード間 | 各ステップの進行方向を示す横方向の矢印 |
| 補足コールアウト | 右列上段 | ステップ完了後の結果や活用例を補足するボックス |
| 注意コールアウト | 右列下段 | 注意事項・制限事項を警告アイコン付きで提示するボックス |
| タイムラインバー | 最下部 | 横線上に関連アイコンを並べ、プロセス全体の連携イメージを補強する |

## Usage Guide（AIへの使い方）

このパターンをAIに指示する際のプロンプト例：

> 「SLIDE-PATTERN-four-step-flow-with-calloutのレイアウトで、〔手順・操作内容〕を4ステップで説明し、右側に補足情報と注意事項を添えたスライドを作成してください。デザインはSLIDE.mdに従ってください。」

**注意点：**
- ステップカードは必ず4つ揃えること。3つや5つにする場合は別パターンの使用を検討する
- コールアウトは上段を「補足・活用例」、下段を「注意・制限事項」の役割で使い分ける
- メインリードコピーはスライドの結論・主張を端的に2行以内で記述する
- タイムラインバーは装飾的要素のため、内容が十分であれば省略可能
