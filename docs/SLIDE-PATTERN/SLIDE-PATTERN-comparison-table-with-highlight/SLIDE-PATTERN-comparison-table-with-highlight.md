# SLIDE-PATTERN-comparison-table-with-highlight

このファイルはスライドのコンテンツエリア（タイトル行より下の領域）のレイアウトパターン定義書です。SLIDE.mdと組み合わせてAIツールに渡すことで、このパターンのスライドを生成できます。タイトルエリア・ページ番号・装飾はSLIDE.mdの `Slide Frame` セクションで定義されるため、このファイルには含みません。

## Overview

**パターン名：** comparison-table-with-highlight
**概要：** 複数のプラン・選択肢を列で比較する表形式レイアウト。ヘッダー行をアクセントカラーで塗りつぶし、特定セルの数値などをハイライトカラーで強調する。
**適したシーン：** 料金プラン比較、サービスプラン説明、仕様比較、機能の有無一覧など複数の選択肢を横断的に比較したいスライド。

## Structure（構造）

コンテンツエリアの中央に横幅いっぱいの比較テーブルを1つ配置する。テーブルは上部にヘッダー行（背景色付き）、下部に各行が交互に薄い背景色で区切られたデータ行で構成される。各セルは縦方向中央揃えで、強調したいデータ（価格など）はアクセントカラーのフォントで表示する。

    structure:
      layout: single
      table:
        header_row: true
        header_bg: accent-color
        row_count: 3
        col_count: 5
        alternating_row_bg: true
        highlight_cells: accent-color-text

## Elements（各要素の役割）

| 要素 | 配置 | 役割 |
|---|---|---|
| テーブルヘッダー行 | テーブル最上部 | 各列の項目名を表示。背景をアクセントカラーで塗りつぶし、白文字で視認性を確保する |
| 行ラベルセル（第1列） | 各データ行の左端 | 比較対象の名称（プラン名など）を表示。薄いグレー背景で他の列と区別する |
| 通常データセル | ヘッダー以外の各セル | テキストや数値などのデータを表示。背景は白または薄いグレーを交互に使う |
| ハイライトセル | 強調したいデータセル | 価格・数値など重要な値をアクセントカラーの大きめフォントで強調表示する |
| テーブル外枠・罫線 | セル境界 | 軽量なボーダーで各セルを区切り、可読性を高める |

## Usage Guide（AIへの使い方）

このパターンをAIに指示する際のプロンプト例：

> 「SLIDE-PATTERN-comparison-table-with-highlightのレイアウトで、3つのサービスプランを料金・対応範囲・納期・おすすめ対象の4軸で比較するスライドを作成してください。デザインはSLIDE.mdに従ってください。」

**注意点：**
- 列数が増えすぎると各セルが狭くなるため、比較軸は4〜6列程度に収める
- ハイライトする値（価格・数値）は1〜2列に絞り、強調が分散しないようにする
- テーブルの縦幅が大きくなる場合は行の高さを均等に調整してコンテンツエリアに収める
- ヘッダー背景色はSLIDE.mdのアクセントカラーに合わせる
