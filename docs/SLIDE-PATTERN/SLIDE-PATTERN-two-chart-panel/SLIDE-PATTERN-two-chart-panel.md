# SLIDE-PATTERN-two-chart-panel
このファイルはスライドのコンテンツエリア（タイトル行より下の領域）のレイアウトパターン定義書です。タイトルエリアを除いたコンテンツ領域のみを対象とし、スライド全体のテンプレートではありません。

## Overview

コンテンツエリアを左右2つのグラフパネルに分割し、各パネルにタイトルバンド・グラフ本体・凡例を配置するデータ比較レイアウト。2種類のグラフデータを対比・並置して見せる場合に適している。

- **適用場面:** 期間比較、カテゴリ別分析、A/B比較、部門別レポート
- **情報密度:** 中〜高（パネル単位で完結した情報を2セット提供）
- **方向性:** 2つの独立データの並列比較

## Structure

```
┌────────────────────────────────────────────────┐
│ タイトルエリア（共通）                           │
├────────────────────────────────────────────────┤
│ ┌──────────────────┐  ┌──────────────────┐      │
│ │ パネルタイトル1   │  │ パネルタイトル2   │      │
│ ├──────────────────┤  ├──────────────────┤      │
│ │                  │  │                  │      │
│ │  [ドーナツ]  凡例  │  │ [ドーナツ] [ドーナツ]│     │
│ │                  │  │                  │      │
│ └──────────────────┘  └──────────────────┘      │
└────────────────────────────────────────────────┘
     左パネル flex:1         右パネル flex:1
```

## Elements

### パネル共通構造
| 要素 | 説明 |
|------|------|
| パネル外枠 | border:1px solid #E0E0E0 / border-radius:4px |
| ヘッダー | padding:8px 16px / background:#F5F5F5 |
| タイトル | font-size:13px / bold / color:#333 |
| パネル本体 | padding:16px / display:flex / gap:12px |

### 左パネル（ドーナツ＋凡例リスト）
| 要素 | 説明 |
|------|------|
| ドーナツSVG | width:100px / height:100px |
| 凡例リスト | font-size:11px / color:#555 /縦並び |
| 凡例ドット | 10×10px / 各カテゴリ色 |
| 各凡例行 | ラベル＋数値（bold） |

### 右パネル（ドーナツ2点横並び）
| 要素 | 説明 |
|------|------|
| ドーナツSVG×2 | 各width:90px / height:90px |
| 各グラフラベル | font-size:11px / bold / color:#555 |
| 配置 | 横並び（display:flex / gap:16px） |

## Usage Guide

- 左右パネルは独立したトピックでもよいが、同じ軸（例：今期vs前期）で対比させると説得力が増す
- 左パネルのドーナツは4〜5分割、右パネルのドーナツは2〜3分割程度が見やすい
- パネルヘッダーのタイトルは10文字以内に収めると見栄えが良い
- 凡例リストは4〜5項目が上限。それ以上はグラフ側でグルーピングを検討する
- パネル間のgapは20pxを維持することでパネル独立性を視覚的に示す
