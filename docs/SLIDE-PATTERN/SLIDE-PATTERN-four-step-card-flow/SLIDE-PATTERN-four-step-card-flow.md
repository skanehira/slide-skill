# SLIDE-PATTERN-four-step-card-flow

このファイルはスライドのコンテンツエリア（タイトル行より下の領域）のレイアウトパターン定義書です。SLIDE.mdと組み合わせてAIツールに渡すことで、このパターンのスライドを生成できます。タイトルエリア・ページ番号・装飾はSLIDE.mdの `Slide Frame` セクションで定義されるため、このファイルには含みません。

## Overview

**パターン名：** four-step-card-flow
**概要：** 上部のリード文と概要説明を配置し、中央にドット付きタイムライン線を横断させた上で、4つの角丸カードを横並びに配置するフローレイアウト
**適したシーン：** 商品開発・プロジェクト・業務フローなど4段階のプロセスを視覚的に示すスライド。各ステップに「STEP番号」「ステップ名」「箇条書き詳細」を持つ情報量が多い手順説明に適する

## Structure（構造）

コンテンツエリアはリード文（短い概要説明）エリアを上部に置き、その下に「タイムラインドット付き横線 ＋ 4カード横並び」の構造を配置する。タイムライン線は4枚のカードの上端付近を横断し、各カード上部のドットで接続位置を示す。カード内はSTEPラベル・大きな番号・ステップ名・区切り線・箇条書き詳細の構成。

    structure:
      layout: flow
      lead_text: top
      timeline:
        type: horizontal-dot-line
        position: above-cards
      card_area:
        direction: horizontal
        card_count: 4
        card_style: rounded-box
        card_structure:
          - step_label
          - step_number_large
          - step_name
          - divider
          - bullet_list

## Elements（各要素の役割）

| 要素 | 配置 | 役割 |
|---|---|---|
| リード文 | コンテンツエリア上部 | スライドの背景・文脈を1〜2行で説明するテキスト |
| タイムラインドット | 各カード上部中央 | カードとタイムライン横線の接続点を示す円形ドット |
| タイムライン横線 | 4カードを横断 | ステップの時系列的なつながりを示す細い水平線 |
| STEPラベル | カード内上段 | 「STEP」の文字をアクセントカラーで表示するラベル |
| ステップ番号（大） | カード内中上段 | 「01」「02」のように大きく番号を表示して視認性を高める |
| ステップ名 | カード内中段 | そのステップの端的な名称を中サイズで表示 |
| 区切り線 | ステップ名とリストの間 | カード内のセクションを視覚的に分離する細い横線 |
| 箇条書き詳細 | カード内下段 | 各ステップの具体的な実施項目を2〜3行で列挙 |

## Usage Guide（AIへの使い方）

このパターンをAIに指示する際のプロンプト例：

> 「SLIDE-PATTERN-four-step-card-flowのレイアウトで、〔プロセス名〕を4ステップで説明してください。各ステップにSTEP番号・ステップ名・2〜3個の詳細箇条書きを入れてください。デザインはSLIDE.mdに従ってください。」

**注意点：**
- カードは必ず4枚揃えること。枚数が変わる場合はカード幅比率を調整する
- タイムラインドットは各カードの上部中央に配置し、横線との接続を視覚的に示す
- STEPラベルと番号はアクセントカラーで統一し、全カードで同じスタイルを維持する
- 箇条書きは「＞」や「▶」などの記号を使うと方向性を強調できる
- リード文はプロセスの意義・背景を1〜2文で記述し、カード群の導入として機能させる
