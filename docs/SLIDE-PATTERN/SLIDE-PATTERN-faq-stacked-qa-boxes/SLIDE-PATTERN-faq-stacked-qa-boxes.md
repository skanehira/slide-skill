# SLIDE-PATTERN-faq-stacked-qa-boxes

このファイルはスライドのコンテンツエリア（タイトル行より下の領域）のレイアウトパターン定義書です。SLIDE.mdと組み合わせてAIツールに渡すことで、このパターンのスライドを生成できます。タイトルエリア・ページ番号・装飾はSLIDE.mdの `Slide Frame` セクションで定義されるため、このファイルには含みません。

## Overview

**パターン名：** faq-stacked-qa-boxes
**概要：** Q&Aのペアをひとつのボックスにまとめ、複数ボックスをコンテンツエリアに縦に積み重ねるFAQレイアウト。「Q.」の質問文はアクセントカラーで強調し、「A.」の回答文はその下にやや大きめのフォントで続ける。
**適したシーン：** よくある質問（FAQ）の紹介、疑問点の解消、サービスに関するQ&A、想定質問への事前回答など、問いと答えをセットで伝えるスライド。

## Structure（構造）

コンテンツエリア全体を縦方向に均等分割し、各Q&Aペアをひとつの薄い背景色ボックスに収めて縦積みする。各ボックス内はQとAの2行構成で、Q行（質問テキスト）はアクセントカラーで小さめに表示し、A行（回答テキスト）はその下に黒テキストで表示する。ボックス間には適切な余白を確保する。

    structure:
      layout: single
      items: stacked-qa-boxes
      item_count: 3
      item:
        q_row: accent-color-label + question-text
        a_row: body-text

## Elements（各要素の役割）

| 要素 | 配置 | 役割 |
|---|---|---|
| Q&Aボックス | コンテンツエリアに縦積み | Q&Aペアをまとめるコンテナ。薄い背景色と角丸ボーダーで視覚的に区切る |
| Q行プレフィクス | ボックス内上部左端 | 「Q.」というラベルをアクセントカラーで表示し、質問であることを明示する |
| 質問テキスト | Q行プレフィクスの右横 | 質問文を太字またはアクセントカラーのフォントで表示する |
| A行テキスト | ボックス内Q行の下 | 回答文を通常フォントで表示する。複数行にまたがってよい |

## Usage Guide（AIへの使い方）

このパターンをAIに指示する際のプロンプト例：

> 「SLIDE-PATTERN-faq-stacked-qa-boxesのレイアウトで、サービスに関するよくある質問3つとその回答をボックス形式で縦に並べるFAQスライドを作成してください。デザインはSLIDE.mdに従ってください。」

**注意点：**
- ボックス数は2〜4個が最適。5個以上はボックスが小さくなりすぎるため、スライドを分割する
- 質問文は1行に収まる短さにし、回答文も2〜3行以内に収める
- Q行のアクセントカラーはSLIDE.mdのブランドカラーに合わせる
- ボックスの高さは内容量に応じて可変にしてよいが、コンテンツエリアに収めること
