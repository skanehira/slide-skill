# SLIDE-PATTERN-key-message-single

このファイルはスライドのコンテンツエリア（タイトル行より下の領域）のレイアウトパターン定義書です。SLIDE.mdと組み合わせてAIツールに渡すことで、このパターンのスライドを生成できます。タイトルエリア・ページ番号・装飾はSLIDE.mdの `Slide Frame` セクションで定義されるため、このファイルには含みません。

## Overview

**パターン名：** key-message-single
**概要：** 1つの大きなキーメッセージをコンテンツエリア中央に配置する強調スライド。上下に細い区切り線を置き、さらに下部に補足テキストを表示する。余白が多くシンプルで印象に残りやすい。
**適したシーン：** 重要な結論・発見・主張を強調する場面、章の締めくくり、印象付けたい1枚

## Structure（構造）

コンテンツエリアを縦方向中央揃えで使用する。上区切り線→キーメッセージ→下区切り線→補足テキストの順で縦に積み上げる。

    structure:
      layout: content-area-centered
      elements:
        - divider-line-top (short, #CCCCCC, 60px)
        - key-message (large, bold, centered, #333)
        - divider-line-bottom (short, #CCCCCC, 60px)
        - supplemental-text (small, muted, #888)

## Elements（各要素の役割）

| 要素 | 配置 | 役割 |
|---|---|---|
| 上部区切り線 | キーメッセージの上 | 視線をメッセージに集中させる装飾的なセパレータ |
| キーメッセージ | コンテンツエリア中央 | 最も伝えたい1文を大きく太く表示 |
| 下部区切り線 | キーメッセージの下 | メッセージの終わりを示す装飾的なセパレータ |
| 補足テキスト | 下部区切り線の下 | キーメッセージの根拠・出典・補足説明を小さく表示 |

## Usage Guide（AIへの使い方）

このパターンをAIに指示する際のプロンプト例：

> 「SLIDE-PATTERN-key-message-singleのレイアウトで、キーメッセージ「[キーメッセージ]」、補足「[補足テキスト]」のスライドを作成してください。デザインはSLIDE.mdに従ってください。」

**注意点：**
- キーメッセージは1文（最大2行）に収めること。長いと効果が薄れる
- 補足テキストは省略してもよい。出典・データ根拠などを入れると信頼性が上がる
- 背景に淡いブランドカラーを敷くことで、前後スライドとのコントラストを強調できる
- キーメッセージは「〜である」「〜が重要だ」など断言形にすることで印象が強まる
