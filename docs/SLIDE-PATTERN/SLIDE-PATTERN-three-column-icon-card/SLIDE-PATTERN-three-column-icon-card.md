# SLIDE-PATTERN-three-column-icon-card

このファイルはスライドのコンテンツエリア（タイトル行より下の領域）のレイアウトパターン定義書です。SLIDE.mdと組み合わせてAIツールに渡すことで、このパターンのスライドを生成できます。タイトルエリア・ページ番号・装飾はSLIDE.mdの `Slide Frame` セクションで定義されるため、このファイルには含みません。

## Overview

**パターン名：** three-column-icon-card
**概要：** 3つの項目を等幅カードで横並びにし、各カードの上部にアイコン・下部に見出し・本文を配置する3列アイコンカードレイアウト
**適したシーン：** サービスの3つの特徴・強み・メリット、3つのターゲット、3つのステップ説明

## Structure（構造）

コンテンツエリアを3等分した横並びカードで構成する。各カードは上部にアイコン（図形またはシンボル）、中央に見出し（H2）、下部に本文テキスト（箇条書きまたは段落）を配置する。カードは背景色または枠線で区切る。

    structure:
      layout: three-equal-columns
      card:
        top:
          elements: [icon (symbol or image)]
        middle:
          elements: [heading-h2]
        bottom:
          elements: [body-text or bullet-list]

## Elements（各要素の役割）

| 要素 | 配置 | 役割 |
|---|---|---|
| アイコン | 各カード上部中央 | その項目を視覚的に表すシンボル |
| 見出し（H2） | アイコン下 | 項目のキーワード・タイトル |
| 本文 | 見出し下 | 詳細説明または箇条書き（2〜4行） |
| カード枠/背景 | 各カード全体 | 3項目の区切りと統一感を作る |

## Usage Guide（AIへの使い方）

このパターンをAIに指示する際のプロンプト例：

> 「SLIDE-PATTERN-three-column-icon-cardのレイアウトで、3つの[テーマ]（[項目1]・[項目2]・[項目3]）をアイコン付きカードで説明してください。デザインはSLIDE.mdに従ってください。」

**注意点：**
- アイコンは実際の画像・SVGを使うか、テキストシンボル（★・✓・→など）で代用できる
- 3列固定のパターン。2列または4列にしたい場合は別パターンを使用
- カード間の余白を十分にとることで整理された印象になる
