# SLIDE-PATTERN-chart-with-callout

このファイルはスライドのコンテンツエリア（タイトル行より下の領域）のレイアウトパターン定義書です。SLIDE.mdと組み合わせてAIツールに渡すことで、このパターンのスライドを生成できます。タイトルエリア・ページ番号・装飾はSLIDE.mdの `Slide Frame` セクションで定義されるため、このファイルには含みません。

## Overview

**パターン名：** chart-with-callout
**概要：** 上部に本文テキスト、下部左に円グラフ（ラベル付き）、下部右に吹き出しコールアウトボックスを配置するデータ解説レイアウト
**適したシーン：** 統計・調査データの解説、数値を強調したいスライド、データに基づく主張の裏付け

## Structure（構造）

コンテンツエリアは縦2段に分かれる。上段は全幅の本文テキストエリア（2〜3行）、下段は左60%にチャート、右40%に吹き出しコールアウトボックスを配置する2カラム構成。チャートには引き出し線付きのラベルが周囲に配置される。

    structure:
      layout: top-bottom-split
      top:
        width: 100%
        type: text
        role: データのコンテキストや主張を説明する本文（2〜3行）
      bottom:
        layout: two-column
        left:
          width: 60%
          type: chart
          role: 円グラフ（各セグメントに引き出し線＋ラベル付き）
        right:
          width: 40%
          type: callout
          role: 最重要データを吹き出しボックスで強調表示

## Elements（各要素の役割）

※ タイトル（H1）はSLIDE.mdの Slide Frame で定義されるため、この表には含めない。コンテンツエリアの要素のみを記述する。

| 要素 | 配置 | 役割 |
|---|---|---|
| 本文テキスト | 上段・全幅 | スライドの主張・データの背景を2〜3行で説明する導入文 |
| 円グラフ | 下段・左60% | 各カテゴリの割合を視覚化。引き出し線とラベルでセグメントを識別する |
| グラフラベル | 円グラフ周囲 | 各セグメントのカテゴリ名。引き出し線で円グラフと接続する |
| 吹き出しコールアウト | 下段・右40% | 強調したい1つのデータ（例：最大セグメントの意味）を枠囲みで提示する |

## Usage Guide（AIへの使い方）

このパターンをAIに指示する際のプロンプト例：

> 「SLIDE-PATTERN-chart-with-calloutのレイアウトで、[チャートデータとコールアウトに入れるメッセージ]を使ってスライドを作成してください。デザインはSLIDE.mdに従ってください。」

**注意点：**
- 円グラフのラベルは引き出し線（アノテーションライン）で接続すると視認性が高まる
- コールアウトボックスはアクセントカラーのボーダーまたは背景で強調するとメリハリが出る
- チャートが棒グラフや折れ線グラフの場合も同じ構造で適用できる
- 本文テキストは結論・主張を先に書くボトムラインファースト形式が効果的
