# SLIDE-PATTERN-three-col-icon-card-header-band

このファイルはスライドのコンテンツエリア（タイトル行より下の領域）のレイアウトパターン定義書です。SLIDE.mdと組み合わせてAIツールに渡すことで、このパターンのスライドを生成できます。タイトルエリア・ページ番号・装飾はSLIDE.mdの `Slide Frame` セクションで定義されるため、このファイルには含みません。

## Overview

**パターン名：** three-col-icon-card-header-band
**概要：** 3列カードの上部にカラーの帯（ヘッダーバンド）を持ち、六角形などのアイコン・見出し・箇条書きテキスト・CTAボタンで構成されるカードレイアウト。
**適したシーン：** 施策・取り組み・次のアクションを3つ並べて提示するスライド。各項目にKPIや目標数値をボタンで強調したい場合に最適。

## Structure（構造）

コンテンツエリアの上部にリード文を置き、その下に3つのカードを横並びで配置する。各カードは上部のカラーバンド・アイコン・見出し（下線付き）・箇条書き本文・CTAボタンで縦に積み上げられた構造。

    structure:
      layout: three-column-grid
      lead_text: true
      columns: 3
      card:
        header_band: true
        icon: hexagon
        heading: true
        heading_underline: true
        body: bullet-list
        cta_button: true

## Elements（各要素の役割）

| 要素 | 配置 | 役割 |
|---|---|---|
| リード文 | カードグループ上部 | スライド全体の補足説明・文脈を示すテキスト |
| ヘッダーバンド | カード最上部 | カードのアクセントカラー帯（ブランドカラー） |
| アイコン | ヘッダーバンド上に重なる形で中央 | 各カードのテーマを視覚的に示す六角形アイコン |
| 見出し | アイコン直下、中央揃え | カードの主題を示す短いテキスト（下線付き） |
| 箇条書き本文 | 見出し下 | 取り組み内容・施策の具体的な説明を箇条書きで列挙 |
| CTAボタン | カード最下部 | 目標値・KPI・カテゴリラベルをボタン形式で強調表示 |

## Usage Guide（AIへの使い方）

このパターンをAIに指示する際のプロンプト例：

> 「SLIDE-PATTERN-three-col-icon-card-header-bandのレイアウトで、次月の3つの重点施策（各施策の取り組み内容と目標KPI付き）を提示するスライドを作成してください。デザインはSLIDE.mdに従ってください。」

**注意点：**
- ヘッダーバンドはブランドカラーで塗りつぶし、アイコンはバンドに重なるように配置する
- 見出しテキストの直下には必ず下線（ボーダー）を引く
- CTAボタンはカード内で最も目立つ要素として下部に固定配置し、目標値や数値を入れる
- 箇条書きは2〜3項目を目安とし、簡潔に記述する
- 3カードは等幅・等間隔で横並びにする
