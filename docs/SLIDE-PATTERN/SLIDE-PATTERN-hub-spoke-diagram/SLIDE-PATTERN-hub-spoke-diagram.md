# SLIDE-PATTERN-hub-spoke-diagram

このファイルはスライドのコンテンツエリア（タイトル行より下の領域）のレイアウトパターン定義書です。SLIDE.mdと組み合わせてAIツールに渡すことで、このパターンのスライドを生成できます。タイトルエリア・ページ番号・装飾はSLIDE.mdの `Slide Frame` セクションで定義されるため、このファイルには含みません。

## Overview

**パターン名：** hub-spoke-diagram
**概要：** 中央に主テーマを表す図形（円・六角形・多角形など）を配置し、周囲に4〜6つの要素テキストを放射状に配置するハブ＆スポーク構造のレイアウト
**適したシーン：** 中心概念と関連要素の関係説明、PDCAサイクル変形、メリット・リスクの多角的説明

## Structure（構造）

コンテンツエリア中央に大きな図形（ハブ）を配置し、その上下左右または対角に4〜6つのテキストボックス（スポーク）を配置する。スポークには見出し（H3）と説明文を記述する。線でハブとスポークをつなぐことがある。

    structure:
      layout: hub-spoke
      center:
        type: circle or hexagon or polygon
        label: [core-concept-name]
      spokes:
        count: 4-6
        position: radial (top/right/bottom/left or corners)
        spoke:
          elements: [heading-h3, description-body]

## Elements（各要素の役割）

| 要素 | 配置 | 役割 |
|---|---|---|
| ハブ図形 | コンテンツ中央 | 全スポークに共通するコアコンセプトを示す |
| ハブラベル | 図形内 | コアコンセプト名 |
| スポーク見出し | 各スポーク上部 | その要素のキーワード |
| スポーク説明文 | 見出し下 | 2〜4行の詳細説明 |
| 接続線（任意） | ハブ〜スポーク間 | ハブと各要素のつながりを示す |

## Usage Guide（AIへの使い方）

このパターンをAIに指示する際のプロンプト例：

> 「SLIDE-PATTERN-hub-spoke-diagramのレイアウトで、中心に「[中心テーマ]」を置き、周囲に[N]つの要素（[要素1]〜[要素N]）を配置してください。デザインはSLIDE.mdに従ってください。」

**注意点：**
- 4〜6つのスポークが最も視覚的にバランスが良い
- 各スポークのテキスト量は均等にすると全体のバランスが保てる
- ハブ図形と各スポークを細い線でつなぐとより「つながり」が伝わる
