# SLIDE-PATTERN-four-card-2x2

このファイルはスライドのコンテンツエリア（タイトル行より下の領域）のレイアウトパターン定義書です。SLIDE.mdと組み合わせてAIツールに渡すことで、このパターンのスライドを生成できます。タイトルエリア・ページ番号・装飾はSLIDE.mdの `Slide Frame` セクションで定義されるため、このファイルには含みません。

## Overview

**パターン名：** four-card-2x2
**概要：** 4つの項目を2列×2行のカードグリッドに配置し、各カードに番号ヘッダー・見出し・本文を入れるレイアウト
**適したシーン：** 4つの特徴・メリット・変化の説明、2×2マトリクスの各象限の説明

## Structure（構造）

コンテンツエリアを2列×2行の4カードに均等分割する。各カードはヘッダー部（番号バッジ＋見出し）とボディ部（本文テキスト）で構成される。カードはボーダーまたは背景色で区切る。

    structure:
      layout: two-by-two-grid
      columns: 2
      rows: 2
      card:
        header:
          elements: [number-badge, heading]
        body:
          elements: [body-text]

## Elements（各要素の役割）

| 要素 | 配置 | 役割 |
|---|---|---|
| 番号バッジ | 各カードヘッダー左 | 01〜04の番号 |
| 見出し | バッジ右（太字） | カードのキーワード・タイトル |
| 本文 | カードボディ | 見出しの詳細説明（2〜4行） |
| カード枠 | 各カード全体 | 4項目の区切りと読みやすさを確保 |

## Usage Guide（AIへの使い方）

このパターンをAIに指示する際のプロンプト例：

> 「SLIDE-PATTERN-four-card-2x2のレイアウトで、4つの[テーマ]を2×2グリッドで説明してください。各カードに見出しと説明文を入れてください。デザインはSLIDE.mdに従ってください。」

**注意点：**
- 4カード固定のパターン。3項目や5項目には向かない
- 各カードの本文テキスト量を揃えるとバランスが良い
- カードヘッダーの背景色を強調色にすると視覚的にメリハリが出る
