# SLIDE-PATTERN-donut-chart-spoke-labels

このファイルはスライドのコンテンツエリア（タイトル行より下の領域）のレイアウトパターン定義書です。SLIDE.mdと組み合わせてAIツールに渡すことで、このパターンのスライドを生成できます。タイトルエリア・ページ番号・装飾はSLIDE.mdの `Slide Frame` セクションで定義されるため、このファイルには含みません。

## Overview

**パターン名：** donut-chart-spoke-labels
**概要：** スライド中央にドーナツ型円グラフ（分割セグメント）を配置し、各セグメントから放射状に引き出し線を伸ばして左右のテキストパネル（見出し＋本文）に接続するレイアウト。
**適したシーン：** 分類・課題・構成要素を5〜6項目に等分し、円グラフで視覚的バランスを強調しながら各項目を詳しく説明したいとき。

## Structure（構造）

コンテンツエリア中央にドーナツ（リング）グラフを配置し、ドーナツの穴中央にテーマラベルを置く。各セグメントの重心から放射方向に引き出し線を伸ばし、左右に分配したテキストブロック（見出し＋本文）に接続する。左側ブロック群は右側端揃え、右側ブロック群は左側端揃えにする。

    structure:
      layout: radial-donut
      center:
        element: donut-chart
        hole: center-label
        segments: 5
      spokes:
        count: 5
        connector: leader-line
        distribution:
          left: 2
          right: 3
      text_block:
        composition: heading + body-text

## Elements（各要素の役割）

| 要素 | 配置 | 役割 |
|---|---|---|
| ドーナツグラフ | 中央 | 等分セグメントで項目数を視覚化するリング形状の図形 |
| セグメント番号 | 各セグメント内 | 各セクションの連番（01〜N）を表示 |
| 中央ラベル | ドーナツの穴 | テーマ名・合計表記など短いテキストを配置 |
| 引き出し線 | セグメントから外方向 | セグメントとテキストブロックを結ぶ水平または折れ曲がり線 |
| 見出し | 各引き出し線の先端 | 各項目のトピック名（太字・アクセントカラー） |
| 本文テキスト | 見出しの下 | 統計データ・説明・補足（2〜4行程度） |

## Usage Guide（AIへの使い方）

このパターンをAIに指示する際のプロンプト例：

> 「SLIDE-PATTERN-donut-chart-spoke-labelsのレイアウトで、[テーマ]の[課題/要素/分類]を5項目円グラフ形式で配置してください。デザインはSLIDE.mdに従ってください。」

**注意点：**
- セグメントは等分が基本。異なる比率にする場合は各セグメントの割合をAIに明示する
- 引き出し線は折れ曲がり（エルボー型）が整理しやすいが、直線でも成立する
- 左右の本文テキスト量は極力そろえる（片側だけ長くなると視覚バランスが崩れる）
- 5〜6セグメントが最適。7以上はセグメントが細くなり視認性が低下する
