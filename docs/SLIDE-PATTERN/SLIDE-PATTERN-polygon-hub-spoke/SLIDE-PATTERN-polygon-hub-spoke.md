# SLIDE-PATTERN-polygon-hub-spoke

このファイルはスライドのコンテンツエリア（タイトル行より下の領域）のレイアウトパターン定義書です。SLIDE.mdと組み合わせてAIツールに渡すことで、このパターンのスライドを生成できます。タイトルエリア・ページ番号・装飾はSLIDE.mdの `Slide Frame` セクションで定義されるため、このファイルには含みません。

## Overview

**パターン名：** polygon-hub-spoke
**概要：** スライド中央に多角形（ペンタゴン/ヘキサゴン等）を配置し、各頂点に番号付き円バッジを置く。各頂点の外側（左・右・上・下）に説明テキストパネルを放射状に配置するハブ＆スポーク型レイアウト。
**適したシーン：** 相互に連関する5〜6つの要素・メリット・課題を等価に並べて中心テーマとの結びつきを示したいとき。

## Structure（構造）

コンテンツエリアの中央に多角形ダイアグラムを配置し、多角形の各辺に沿って番号バッジを並べる。各バッジから放射方向に説明テキストブロック（見出し＋本文）を接続する。左側2〜3ブロック・右側2〜3ブロックに振り分けることが多い。

    structure:
      layout: radial-polygon
      center:
        element: polygon-shape
        sides: 5
        label: center-theme
      spokes:
        count: 5
        node: circle-badge
        content: heading + body-text
        distribution:
          left: 3
          right: 2

## Elements（各要素の役割）

| 要素 | 配置 | 役割 |
|---|---|---|
| 多角形（ペンタゴン） | 中央 | 全体テーマを内包する図形。辺と頂点が各スポークのアンカーになる |
| 中央ラベル | 多角形の内部 | テーマ名・コンセプト名（短いテキスト）を表示 |
| 番号バッジ（円） | 各頂点 | 連番（01〜N）を入れた小円。頂点と説明テキストをつなぐ中継点 |
| 見出し | 各バッジの外側 | 各スポークのトピック名（色付き短ラベル） |
| 本文テキスト | 見出しの下 | 各スポークの説明・根拠・補足（2〜4行程度） |
| 多角形の辺 | 各頂点をつなぐ線 | 要素間の相互連関を示す |

## Usage Guide（AIへの使い方）

このパターンをAIに指示する際のプロンプト例：

> 「SLIDE-PATTERN-polygon-hub-spokeのレイアウトで、[中心テーマ]の[メリット/要素/課題]を5項目放射状に配置してください。デザインはSLIDE.mdに従ってください。」

**注意点：**
- 多角形の辺数（頂点数）は項目数に合わせる（5項目→ペンタゴン、6項目→ヘキサゴン）
- 各説明テキストブロックは左辺グループ・右辺グループでテキスト揃え方向を変える（左グループは右揃え、右グループは左揃えが読みやすい）
- 多角形は塗りつぶし色を薄くし、テキストの可読性を確保する
- 本文テキストが長い場合は行間を広めにとり、バッジとの間隔を一定に保つ
