# SLIDE-PATTERN-headline-three-donut-charts
このファイルはスライドのコンテンツエリア（タイトル行より下の領域）のレイアウトパターン定義書です。タイトルエリアを除いたコンテンツ領域のみを対象とし、スライド全体のテンプレートではありません。

## Overview

コンテンツエリア上部に大きな見出し2行（キーメッセージ）を配置し、下部に3つのドーナツグラフを横並びで表示する数値インパクトレイアウト。重要なメッセージを文字で強調しながら、その根拠となる3つの数値指標をグラフで視覚化したい場合に適している。

- **適用場面:** エグゼクティブサマリー、成果発表、数値インパクト訴求
- **情報密度:** 中（メッセージ＋グラフ3点の明快な構成）
- **方向性:** 「言葉のインパクト＋数値の裏付け」の組み合わせ

## Structure

```
┌────────────────────────────────────────────────┐
│ タイトルエリア（共通）                           │
├────────────────────────────────────────────────┤
│                                                │
│  ■ 大見出し1行目：キーメッセージのテキスト      │
│  ■ 大見出し2行目：補足メッセージのテキスト      │
│  ─────────────────────────────────────────     │
│                                                │
│   ┌──────────┐  ┌──────────┐  ┌──────────┐   │
│   │  ドーナツ │  │  ドーナツ │  │  ドーナツ │   │
│   │  ( 70%) │  │  ( 55%) │  │  ( 82%) │   │
│   │  見出し  │  │  見出し  │  │  見出し  │   │
│   │  説明   │  │  説明   │  │  説明   │   │
│   └──────────┘  └──────────┘  └──────────┘   │

```

## Elements

### 上部見出しエリア
| 要素 | 説明 |
|------|------|
| エリアラベル | 「KEY MESSAGE」等の小文字ラベル |
| 大見出し（1行目） | font-size:20px / bold / color:#333 |
| 大見出し（2行目） | font-size:20px / bold / color:#333 |
| 区切り線 | height:1px / background:#E0E0E0 |

### 下部グラフエリア（3セル）
| 要素 | 説明 |
|------|------|
| ドーナツSVG | width:110px / height:110px / 2分割配色 |
| 中央数値 | font-size:18px / bold / color:#333 |
| 中央ラベル | font-size:9px / fill:#888 |
| 見出し | font-size:13px / bold / color:#333 |
| 説明テキスト | font-size:11px / color:#666 / center寄せ |
| 凡例 | font-size:10px / color:#999 / flex横並び |

## Usage Guide

- 見出しは「〇〇が△△%改善」「□□において◇◇を達成」のように、具体的な成果を短く記述する
- 3つのドーナツは同じ指標種別（達成率、割合、スコアなど）で統一するとわかりやすい
- 見出し文字サイズは20pxを基準とし、文字数が多い場合は18pxまで縮小可
- 説明テキストは1行に収める。2行になる場合はfont-size:10pxへ調整する
- 凡例はデータ名と残り（「その他」等）の2項目で構成する
