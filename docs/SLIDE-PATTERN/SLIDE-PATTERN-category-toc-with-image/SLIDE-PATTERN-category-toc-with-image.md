# SLIDE-PATTERN-category-toc-with-image

このファイルはスライドのコンテンツエリア（タイトル行より下の領域）のレイアウトパターン定義書です。SLIDE.mdと組み合わせてAIツールに渡すことで、このパターンのスライドを生成できます。タイトルエリア・ページ番号・装飾はSLIDE.mdの `Slide Frame` セクションで定義されるため、このファイルには含みません。

## Overview

**パターン名：** category-toc-with-image
**概要：** カテゴリ見出し（英語大文字＋日本語サブタイトル）と縦線付き箇条書き項目を縦積みした目次を左側に、装飾イメージを右側に配置する2カラムレイアウト
**適したシーン：** ビジュアルを取り入れたモダンな目次スライド、章立てが3〜4カテゴリ程度のプレゼン

## Structure（構造）

コンテンツエリアを左70%・右30%の2カラムに分割する。左カラムにカテゴリ別目次（英語見出し＋日本語サブタイトル＋縦線付き箇条書き）を縦積みで配置し、右カラムには装飾イラストまたは画像を配置する。

    structure:
      layout: two-column
      left:
        width: 70%
        type: category-toc-list
        elements:
          - category-heading (英語大文字）
          - category-subtitle (日本語小文字)
          - bullet-list (縦線付き箇条書き、各項目)
      right:
        width: 30%
        type: image
        role: 装飾イラスト・ビジュアルイメージ（透過・薄め処理が多い）

## Elements（各要素の役割）

※ タイトル（H1）はSLIDE.mdの Slide Frame で定義されるため、この表には含めない。コンテンツエリアの要素のみを記述する。

| 要素 | 配置 | 役割 |
|---|---|---|
| カテゴリ見出し（英語） | 左カラム・各カテゴリの先頭 | セクション名を英語の大きなフォントで表示（例：Company, Business） |
| カテゴリサブタイトル（日本語） | カテゴリ見出しの右横または直下 | カテゴリの内容を日本語で補足説明（小さめフォント） |
| 縦線 | 箇条書き項目の左端 | 各項目を縦に繋ぐ細い縦線。リストの視覚的まとまりを表現 |
| 箇条書き項目 | 左カラム・縦線の右 | 各カテゴリに属するページ・トピック名（丸ポイント付き） |
| 装飾イメージ | 右カラム全体 | ブランドや内容に関連するイラスト・写真（薄め処理で背景的に配置） |

## Usage Guide（AIへの使い方）

このパターンをAIに指示する際のプロンプト例：

> 「SLIDE-PATTERN-category-toc-with-imageのレイアウトで、[カテゴリ名と各カテゴリの項目リスト]を使って目次スライドを作成してください。右側には[テーマに合う装飾イメージ]を配置してください。デザインはSLIDE.mdに従ってください。」

**注意点：**
- 右側の装飾イメージは低透明度（20〜40%程度）にすることで目次テキストが読みやすくなる
- カテゴリ数が2〜4個のときに最もバランスが取れる
- 装飾イメージがない場合は右カラムをカラーブロックやグラデーションで代替できる
- 縦線はborder-leftまたは細い矩形で実装する
