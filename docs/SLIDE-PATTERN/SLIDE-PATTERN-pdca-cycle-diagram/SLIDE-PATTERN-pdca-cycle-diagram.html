Lead Text — スライドの概要・主旨（1〜2文）
PDCAサイクルは仕事と業務改善を継続的に進めるための基本フレームです。まず目標を定めて計画を立て、実行し、結果を検証して改善策を反映します。このフローを活性化させることで成果の再現性を高めていきます。
phase-block / top-left
Action（対策・改善）
Check段階で得られた検証結果から、気付きや課題などを元に改善策を考えていきます。次のPlanサイクルへとつなげる行動を定めます。
phase-block / top-right
Plan（計画）
まず目標や目的を明確に設定し、それらを達成するための具体的な行動計画（アクションプラン）を作成します。
circular cycle diagram
phase-block / bottom-left
Check（評価）
設定した目標やアクションプランに対する達成度や進捗を検証・評価します。結果だけでなく、その結果に至った原因を分析することが重要です。
phase-block / bottom-right
Do（実行）
Planで立てた計画を実行します。「やりっぱなし」にならないよう記録を残し、次のCheckフェーズで振り返れるようにします。
